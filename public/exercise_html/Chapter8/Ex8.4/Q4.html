Question 4.
The probability that atleast one of A and B occur is 0.6. If A and B occur simultaneously with probability 0.2, then find P(\(\overline{\mathbf{A}}\)) + P(\(\overline{\mathbf{B}}\)).
Solution:
P(A∪B) = 0.6
P(A∩B) = 0.2
P(A) + P(B) = [1 – P(A∪B)] + [1 – P(A∩B)] = [1 – 0.6] + [1 – 0.2]
= 0.4 + 0.8 = 1.2
