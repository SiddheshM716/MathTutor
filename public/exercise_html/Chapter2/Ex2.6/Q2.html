How many consecutive odd integers beginning with 5 will sum to 480?
Answer:
First term (a) = 5
Common difference (d) = 2
(consecutive odd integer)
Sn = 480
\(\frac { n }{ 2 } \) [2a + (n-1) d] = 480
\(\frac { n }{ 2 } \) [10 + (n-1)2] = 480

n + 24 = 0 or n – 20 = 0
n = -24 or n = 20
[number of terms cannot be negative]
