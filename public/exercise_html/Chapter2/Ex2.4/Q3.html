Question 3.
Find the nth term of the following sequences
(i) 2, 5, 10, 17, ……
Answer:
(12 + 1);(22 + 1),(32 + 1),(42 + 1)….
nth term is n2 + 1
an = n2 + 1

(ii) 0,\(\frac { 1 }{ 2 } \),\(\frac { 2 }{ 3 } \) ……
Answer:
(\(\frac { 1-1 }{ 1 } \)), (\(\frac { 2-1 }{ 2 } \)), (\(\frac { 3-1 }{ 3 } \)) …..
nth term is \(\frac { n-1 }{ n } \)
an = \(\frac { n-1 }{ n } \)

(iii) 3,8,13,18,…….
Answer:
[5(1) -2], [5(2) – 2], [5(3) – 2], [5(4) – 2] ….
The nth term is 5n – 2
an = 5n – 2
