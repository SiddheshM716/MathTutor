Question 4.
Find x so that x + 6, x + 12 and x + 15 are consecutive terms of a Geometric Progression
Solution:
G.P = x + 6, x + 12, x + 15
In G.P r = \(\frac{t_{2}}{t_{1}}=\frac{t_{3}}{t_{2}}\)
\(\frac{x+12}{x+6}=\frac{x+15}{x+12}\)
(x + 12)2 = (x + 6) (x + 5)
x2 + 24x + 144 = x2 + 6x + 15x + 90
24x – 21x = 90 – 144
3x = -54
x = \(\frac { -54 }{ 3 } \) = -18
x = -18
