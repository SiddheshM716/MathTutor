Question 4.
How many terms of the series 13 + 23 + 33 + …………… should be taken to get the sum 14400?
Answer:
13 + 23 + 33 + ……. + n3 = 14400

\(\frac{n(n+1)}{2}\) = \(\sqrt { 14400 }\)
\(\frac{n(n+1)}{2}\) = 120 ⇒ n2 + n = 240

n2 + n – 240 = 0
(n + 16) (n – 15) = 0
(n + 16) = 0 or (n – 15) = 0
n = -16 or n = 15 (Negative will be omitted)
∴ The number of terms taken is 15
