Question 7.
The perpendicular PS on the base QR of a ∆PQR intersects QR at S, such that QS = 3 SR. Prove that 2PQ2 = 2PR2 + QR2.
Solution:
Given QS = 3SR
QR = QS + SR
= 3SR + SR = 4SR
SR = \(\frac { 1 }{ 4 } \) QR …..(1)
QS = 3SR
SR = \(\frac { QS }{ 3 } \) ……..(2)

From (1) and (2) we get
\(\frac { 1 }{ 4 } \) QR = \(\frac { QS }{ 3 } \)
∴ QS = \(\frac { 3 }{ 4 } \) QR ………(3)
In the right ∆ PQS,
PQ2 = PS2 + QS2 ……….(4)
Similarly in ∆ PSR
PR2 = PS2 + SR2 ………..(5)
Subtract (4) and (5)
PQ2 – PR2 = PS2 + QS2 – PS2 – SR2
= QS2 – SR2

PQ2 – PR2 = \(\frac { 1 }{ 2 } \) QR2
2PQ2 – 2PR2 = QR2
2PQ2 = 2PR2 + QR2
Hence the proved.
