Question 7.
In figure DE || BC and CD || EE Prove that AD2 = AB × AF.

Solution:
Given: In ∆ABC, DE || BC and CD || EF

To Prove: AD2 = AB × AF
Proof: In ∆ABC, DE || BC (Given)
By basic proportionality theorem
\(\frac { AB }{ AD } \) = \(\frac { AC }{ AE } \) ……….. (1)
In ∆ADC; FE || DC (Given)
By basic Proportionality theorem
\(\frac { AD }{ AF } \) = \(\frac { AC }{ AE } \) ……..(2)
From (1) and (2) we get
\(\frac { AB }{ AD } \) = \(\frac { AD }{ AF } \)
AD2 = AB × AF
Hence it is proved
