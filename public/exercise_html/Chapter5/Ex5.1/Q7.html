Question 7.
If the points A(-3, 9), B(a, b) and C(4, -5) are collinear and if a + b = 1, then find a and b.
Solution:
Since the three points are collinear
Area of a ∆ = 0

-3b – 5a + 36 – 9a – 4b – 15 = 0
-7b – 14a + 21 = 0
(÷ by 7) – b – 2a + 3 = 0
2a + b – 3 = 0

Substitute the value of a = 2 in (2) ⇒ 2 + b = 1
b = 1 – 2 = -1
The value of a = 2 and b = -1
