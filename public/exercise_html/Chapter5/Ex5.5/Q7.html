Question 7.
The slope of the line which is perpendicular to a line joining the points (0, 0) and (- 8, 8) is ………..
(1) -1
(2) 1
(3) \(\frac { 1 }{ 3 } \)
(4) -8
Answer:
(2) 1
Hint:
Slope of a line = \(\frac{y_{2}-y_{1}}{x_{2}-x_{1}}\)
= \(\frac { 8-0 }{ -8-0 } \) = \(\frac { 8 }{ -8 } \) = -1
Slope of the Perpendicular = 1
