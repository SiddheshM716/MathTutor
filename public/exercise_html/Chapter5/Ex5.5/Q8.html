Question 8.
If slope of the line PQ is \(\frac{1}{\sqrt{3}}\) then slope of the perpendicular bisector of PQ is …………..
(1) \(\sqrt { 3 }\)
(2) –\(\sqrt { 3 }\)
(3) \(\frac{1}{\sqrt{3}}\)
(4) 0
Answer:
(2) –\(\sqrt { 3 }\)
Hint:
Slope of a line = \(\frac{1}{\sqrt{3}}\)
Slope of the ⊥r bisector = –\(\sqrt { 3 }\)
