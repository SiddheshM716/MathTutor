Question 10.
The equation of a line passing through the origin and perpendicular to the line lx -3y + 4 = 0 is
(1) 7x – 3y + 4 = 0
(2) 3x – 7y + 4 = 0
(3) 3x + 7y = 0
(4) 7x – 3y = 0
Answer:
(3) 3x + 7y = 0
Hint:
Slope of the line of 7x – 3y + 4 = 0
= \(\frac { -7 }{ -3 } \) = \(\frac { 7 }{ 3 } \)
Slope of its ⊥r = \(\frac { -3 }{ 7 } \)
The line passes through (0,0)
Equation of a line is
y – y1 = m(x – x1)
y – 0 = \(\frac { -3 }{ 7 } \) (x – 0)
y = \(\frac { -3 }{ 7 } \) x ⇒ 7y = -3x
3x + 7y = 0
