Question 3.
Find the equation of a line whose inclination is 30° and making an intercept -3 on the Y axis.
Solution:
Angle of inclination = 30°
Slope of a line = tan 30°
(m) = \(\frac{1}{\sqrt{3}}\)
y intercept (c) = -3
Equation of a line is y = mx + c
y = \(\frac{1}{\sqrt{3}}\) x – 3
\(\sqrt { 3 }\) y = x – 3 \(\sqrt { 3 }\)
∴ x – \(\sqrt { 3 }\) y – 3 \(\sqrt { 3 }\) = 0
