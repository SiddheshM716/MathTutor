Question 2.
The equation of a straight line is 2(x – y) + 5 = 0. Find its slope, inclination and intercept on the Y axis.
Solution:
Equation of a line is
2 (x – y) + 5 = 0
2 x – 2y + 5 = 0
-2y = -2x – 5
2y = 2x + 5
y = \(\frac { 2x }{ 2 } \) + \(\frac { 5 }{ 2 } \)
y = x + \(\frac { 5 }{ 2 } \)
Slope of line = 1
Y intercept = \(\frac { 5 }{ 2 } \)
tan θ = 1
tan θ = tan 45°
∴ angle of inclination = 45°
