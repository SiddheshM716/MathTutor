Question 13.
Find the intercepts made by the following lines on the coordinate axes.
(i) 3x – 2y – 6 = 0
(ii) 4x + 3y + 12 = 0
Solution:
(i) 3x – 2y – 6 =0.
x intercept when y = 0
⇒ 3x – 6 = 0
⇒ x = 2
y intercept when x = 0
⇒ 0 – 2y – 6 = 0
⇒ y = -3
(ii) 4x + 3y + 12 = 0. x intercept when y = 0
⇒ 4x + 0 + 12 = 0
⇒ x = -3
y intercept when x = 0
⇒ 0 + 3y + 12 = 0
⇒ y = -4
