Question 3.
A Relation R is given by the set {(x, y)/y = x + 3, x ∈ {0, 1, 2, 3, 4, 5}}. Determine its domain and range.
Answer:
x = {0, 1, 2, 3, 4, 5}
y = x + 3
when x = 0 ⇒ y = 0 + 3 = 3
when x = 1 ⇒ y = 1 + 3 = 4
when x = 2 ⇒ y = 2 + 3 = 5
when x = 3 ⇒ y = 3 + 3 = 6
when x = 4 ⇒ y = 4 + 3 = 7
when x = 5 y = 5 + 3 = 8
R = {(0, 3) (1,4) (2, 5) (3, 6) (4, 7) (5, 8)}
Domain = {0, 1, 2, 3, 4, 5}
Range = {3, 4, 5, 6, 7, 8}
