Question 9.
If the function f is defined by

find the value of
(i) f(3)
(ii) f(0)
(iii) f(1. 5)
(iv) f(2) + f(-2)
Answer:
f(x) = x + 2 when x = {2,3,4,……}
f(x) = 2
f(x) = x – 1 when x = {-2}
(i) f(x) = x + 2
f(3) = 3 + 2 = 5

(ii) f(x) = 2
f(0) = 2

(iii) f(x) = x – 1
f(-1.5) = -1.5 – 1 = -2.5

(iv) f(x) = x + 2
f(2) = 2 + 2 = 4
f(x) = x – 1
f(-2) = – 2 – 1 = – 3
f(2) + f(-2) = 4 – 3
= 1
