Question 9.
Let A = {1, 2, 3, 4} and B = {4, 8, 9, 10}.
A function f: A → B given by f = {(1, 4), (2, 8),(3,9),(4,10)} is a ……………
(1) Many-one function
(2) Identity function
(3) One-to-one function
(4) Into function
Answer:
(3) One-to-one function
Hint:

Different elements of A has different images in B.
∴ It is one-to-one function.
