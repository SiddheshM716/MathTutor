Question 2.
A = {a, b, p}, B = {2, 3}, C = {p, q, r, s} then n[(A ∪ C) × B] is
(1) 8
(2) 20
(3) 12
(4) 16
Answer:
(3) 12
Hint:
A = {a, b, p}, B = {2, 3}, C = {p, q, r, s}
n (A ∪ C) × B
A ∪ C = {a, b, p, q, r, s}
(A ∪ C) × B = {{a, 2), (a, 3), (b, 2), (b, 3), (p, 2), (p, 3), (q, 2), (q, 3), (r, 2), (r, 3), (s, 2), (s, 3)
n [(A ∪ C) × B] = 12
