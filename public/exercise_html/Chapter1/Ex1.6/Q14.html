Question 14.
If g= {(1,1),(2,3),(3,5),(4,7)} is a function given by g(x) = αx + β then the values of α and β are
(1) (-1,2)
(2) (2,-1)
(3) (-1,-2)
(4) (1,2)
Answer:
(2) (2, -1)
Hint: g (x) = αx + β
g(1) = α(1) + β
1 = α + β ….(1)
g (2) = α (2) + β
3 = 2α + β ….(2)
Solve the two equations we get
α = 2, β = -1
