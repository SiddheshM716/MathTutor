Question 5.
Given A = {1,2,3}, B = {2,3,5}, C = {3,4} and D = {1,3,5}, check if
(A ∩ C) × (B ∩ D) = (A × B) ∩ (C × D) is true?

(A ∩ C) × (B ∩ D) = (A × B) ∩ (C × D)
