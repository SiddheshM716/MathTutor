Question 6.
Let A = {x ∈ W | x < 2}, B = {x ∈ N |1 < x < 4} and C = {3, 5}. Verify that
(i) A × (B ∪ C) = (A × B) ∪ (A × C)
(ii) A × (B ∩ C) = (A × B) ∩ (A × C)
(iii) (A ∪ B) × C = (A × C) ∪ (B × C)
(iv) A × (B ∪ C) = (A × B) ∪ (A × C)
Solution:
A = {x ∈ W|x < 2} = {0,1}
B = {x ∈ N |1 < x < 4} = {2,3,4}
C = {3,5}
LHS =A × (B ∪ C)
B ∪ C = {2, 3, 4} ∪ {3, 5}
= {2, 3, 4, 5}
A × (B ∪ C) = {(0, 2), (0, 3), (0,4), (0, 5), (1, 2) , (1, 3), (1, 4),(1, 5)} ………. (1)
RHS = (A × B) ∪ (A × C)
(A × B) = {(0, 2), (0, 3), (0, 4), (1, 2), (1, 3), (1, 4)}
(A × C) = {(0, 3), (0, 5), (1, 3), (1, 5)}
(A × B) ∪ (A × C)= {(0, 2), (0, 3), (0,4), (1, 2), (1, 3), (1, 4), (0, 5), (1, 5)} ….(2)
(1) = (2), LHS = RHS
Hence it is proved.

(ii) A × (B ∩ C) = (A × B) ∩ (A × C)
LHS = A × (B ∩ C)
(B ∩ C) = {3}
A × (B ∩ C) = {(0, 3), (1, 3)} …(1)
RHS = (A × B) ∩ (A × C)
(A × B) = {(0, 2), (0, 3), (0, 4), (1, 2), (1, 3), (1, 4)}
(A × C) = {(0, 3), (0, 5), (1, 3), (1, 5)}
(A × B) ∩ (A × C) = {(0, 3), (1, 3)} ……….. (2)
(1) = (2) ⇒ LHS = RHS.
Hence it is verified.

(iii) (A ∪ B) × C = (A × C) ∪ (B × C)
LHS = (A ∪ B) × C
A ∪ B = {0, 1, 2, 3, 4}
(A ∪ B) × C = {(0, 3), (0, 5), (1, 3), (1, 5), (2, 3), (2, 5), (3, 3), (3, 5), (4, 3), (4, 5)} …………. (1)
RHS = (A × C) ∪ (B × C)
(A × C) = {(0, 3), (0, 5), (1, 3), (1, 5)}
(B × C) = {(2, 3), (2, 5), (3, 3), (3, 5), (4, 3), (4, 5)}
(A × C) ∪ (B × C) = {(0, 3), (0, 5), (1, 3), (1, 5), (2, 3), (2, 5), (3, 3), (3, 5), (4, 3), (4, 5)} ………… (2)
(1) = (2)
∴ LHS = RHS. Hence it is verified.
