Question 2.
Find the LCM of each pair of the following polynomials
(i) a2 + 4a – 12, a2 – 5a + 6 whose GCD is a – 2
Answer:
p(x) = a2 + 4a – 12
= a2 + 6a – 2a – 12
= a (a + 6) – 2(a + 6)
= (a + 6) (a – 2)

g(x) = a2 – 5a + 6
= a2 – 3a – 2a + 6

= a(a – 3) – 2 (a – 3)
= (a – 3) (a – 2)

(ii) x4 – 27a3x, (x – 3a)2 whose GCD is (x – 3a)
Answer:
p(x) = x4 – 27a3x = x[x3 – 27a3]
= x[x3 – (3a)3]
= x(x – 3a) (x2 + 3ax + 9a2)
g(x) = (x – 3a)2
G.C.D. = x – 3a

L.C.M. = x (x – 3a)2 (x2 + 3ax + 9a2)
