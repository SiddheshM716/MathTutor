Question 1.
Find the LCM and GCD for the following and verify that f(x) × g(x) = LCM × GCD

(i) 21x2y, 35xy2
Answer:
p(x) = 21 x2y = 3 × 7 × x2 × y
g(x) = 35xy2 = 5 × 7 × x × y2
G.C.D = 7 xy
L.C.M = 3 × 5 × 7 x2 × y2
= 105 x2y2
L.C.M × G.C.D = 105x2y2 × 7xy
= 735 x3y3 ….(1)
p(x) × g(x) = 21x2y × 35xy2
= 735x3y3 ….(2)
From (1) and (2) we get
L.C.M × G.C.D. = p(x) × g(x)

(ii) (x3 – 1)(x + 1),(x3 + 1)
Answer:
p(x) = (x3 – 1) (x + 1) = (x – 1) (x2 + x + 1) (x + 1)
g(x) = x3 + 1 = (x + 1) (x2 – x + 1)
G.C.D = (x + 1)
L.C.M = (x + 1) (x – 1) (x2 + x + 1) (x2 – x + 1)
L.C.M × G.C.D = (x + 1) (x – 1)(x2 + x + 1)(x2 – x + 1)x(x + 1)
= (x + 1)2 (x – 1) (x2 + x + 1) (x2 – x + 1) ……….(1)
p(x) × g(x) = (x – 1) (x2 + x + 1) (x + 1) (x + 1) (x2 – x + 1)
= (x + 1)2 (x – 1) (x2 + x + 1) (x2 – x + 1) ……….(2)
From (1) and (2) we get
L.C.M × G.C.D. = p(x) × g(x)

(iii) (x2y + xy2), (x2 + xy)
Answer:
p(x) = x2y + xy2 = xy(x + y)
g(x) = x2 + xy = x(x + y)
G.C.D = x(x+y)
L.C.M = xy (x +y).
L.C.M × G.C.D = xy(x + y) × x(x + y)
= x2y(x + y)2 …..(1)
p(x) × g(x) = xy(x + y) × x(x + y)
= x2y(x + y)2
From (1) and (2) we get
L.C.M × G.C.D. = p(x) × g(x)
