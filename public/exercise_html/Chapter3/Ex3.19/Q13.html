Question 13.
The number of points of intersection of the quadratic polynomial x2 + 4x + 4 with the X axis is …………….
(1) 0
(2) 1
(3) 0 or 1
(4) 2
Answer:
(2) 1
Hint:
(x + 2)2 = 0 ⇒ (x + 2) (x + 2) = 0
x + 2 = 0 or x + 2 = 0 ⇒ x = -2 or x = -2
Number of points of intersection is 1 (both the values are same)
