Question 20.

Which of the following statements are correct?

(1) (i) and (ii) only
(2) (ii) and (iii) only
(3) (iii) and (iv)
(4) all of these
Answer:
(1) (i) and (ii) only
Hint.
(i) AB + C : order of A = 2 × 3
order of B = 3 × 2
order of AB = 2 × 2
order of C = 2 × 2
It is possible to add AB + C

(ii) BC :
order of B = 3 × 2
order of C = 2 × 2
It is possible to find BC

(iii) BA is not possible
order of B = 3 × 2
order of A = 3 × 2
BA does not exist.
BA + C is not a correct statement

(iv) ABC is not possible
order of A = 2 × 3 order of
B = 3 × 2
order of AB = 2 × 2
order of C = 3 × 2
It is not possible to multiply AB and C.
∴ The statement ABC is not correct.
