Question 3.
Find the values of a and b if the following polynomials are perfect squares.
(i) 4x4 – 12x3 + 37x2 + bx + a
Answer:

Since it is a perfect square
b + 42 = 0
b = – 42
a – 49 = 0
a = 49
∴ The value of a = 49 and b = – 42

(ii) ax4 + bx3 + 361x2 + 220x + 100

100 + 220x + 361x2 + bx3 + ax4

b – 264 = 0
