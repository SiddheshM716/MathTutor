Question 2.
Simplify
(i) \(\frac{x+4}{3 x+4 y} \times \frac{9 x^{2}-16 y^{2}}{2 x^{2}+3 x-20}\)
Answer:
9x2 – 16y2 = (3x)2 – (4y)2
= (3x + 4y) (3x – 4y)
2x2 + 3x – 20 = 2x2 + 8x – 5x – 20
= 2x (x + 4) – 5 (x + 4)
= (x + 4) (2x – 5)

(ii) \(\frac{x^{3}-y^{3}}{3 x^{2}+9 x y+6 y^{2}} \times \frac{x^{2}+2 x y+y^{2}}{x^{2}-y^{2}}\)
Answer:
x3 – y3 = (x – y) (x2 + xy + y2)
x2 + 2xy + y2 = (x + y) (x + y)

3x2 + 9xy + 6y2 = 3(x2 + 3xy + 2y2)
= 3 (x + 2y) (x + y)
(x2 – y2) = (x + y) (x – y)
