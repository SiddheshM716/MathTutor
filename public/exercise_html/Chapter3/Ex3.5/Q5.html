Question 5.
If a polynomial p(x) = x2 – 5x – 14 when divided by another polynomial q(x) gets reduced to \(\frac { x-7 }{ x+2 } \) find q(x).
Answer:
p(x) = x2 – 5x – 14
= (x – 7) (x + 2)

By the given data
\(\frac { p(x) }{ q(x) } \) = \(\frac { (x-7) }{ x+2 } \)
\(\frac{(x-7)(x+2)}{q(x)}\) = \(\frac { (x-7) }{ x+2 } \)
q(x) × (x – 7) = (x – 7) (x + 2) (x + 2)
q(x) = \(\frac{(x-7)(x+2)(x+2)}{(x-7)}\)
= (x + 2)2
