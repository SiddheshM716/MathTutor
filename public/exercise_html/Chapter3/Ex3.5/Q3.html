Simplify
(i) \(\frac{2 a^{2}+5 a+3}{2 a^{2}+7 a+6} \div \frac{a^{2}+6 a+5}{-5 a^{2}-35 a-50}\)
Answer:

2 a2 + 5a + 3a + 3 = 2a2 + 2a + 3a + 3
= 2a(a + 1) + 3 (a + 1)
= (a + 1) (2a + 3)

2a2 + 7a + 6 = 2a2 + 3a + 4a + 6
= a(2a + 3) + 2 (2a + 3)
= (2a + 3) (a + 2)

a2 + 6a + 5 = (a + 5) + (a + 1)
-5a2 – 35a – 50 = -5(a2 + 7a + 10)
= -5(a + 5)(a + 2)

(ii) \(\frac{b^{2}+3 b-28}{b^{2}+4 b+4}+\frac{b^{2}-49}{b^{2}-5 b-14}\)
Solution:

b2 + 3b – 28 = (b + 7) (b – 4)
b2 + 4b + 4 = (b + 2) (b + 2)
b2 – 49 = b2 – 72
= (b + 7) (b – 7)
b2 – 5b – 14 = (b – 7) (b + 2)

x2 – x – 6 = (x – 3) (x + 2)

(iv) \(\frac{12 t^{2}-22 t+8}{3 t} \div \frac{3 t^{2}+2 t-8}{2 t^{2}+4 t}\)
Answer:
12t2 – 22t + 8 = 2(6t2 – 11t + 4)
= 2[6t2 – 8t – 3t + 4]

= 2[2t (3t – 4) – 1 (3t – 4)]
= 2(3t – 4) (2t – 1)
3t2 + 2t – 8 = 3t2 + 6t – 4t – 8
= 3t(t + 2) – 4 (t + 2)

= (t + 2) (3t – 4)
2t2 + 4t = 2t(t + 2)
