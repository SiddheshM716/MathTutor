P and Q are the mid-points of the sides CA and CB respectively of a ΔABC, right angled at C. Prove that 4(AQ² + BP²) = 5AB².

1

Identify the right triangles and apply the Pythagorean theorem.

For ΔAQC (right-angled at C):

AQ² = AC² + QC² ... (1)

For ΔBPC (right-angled at C):

BP² = BC² + CP² ... (2)

For ΔABC (right-angled at C):

AB² = AC² + BC² ... (3)

2

Add equations (1) and (2) to combine the terms.

AQ² + BP² = AC² + QC² + BC² + CP²

3

Multiply both sides by 4 and simplify using mid-point properties (QC = CB/2, CP = CA/2).

4(AQ² + BP²) = 4AC² + 4QC² + 4BC² + 4CP²

= 4AC² + (2QC)² + 4BC² + (2CP)²

= 4AC² + BC² + 4BC² + AC²

= 5AC² + 5BC²

= 5(AC² + BC²)

From equation (3), substitute AB² for (AC² + BC²):

4(AQ² + BP²) = 5AB²

4(AQ² + BP²) = 5AB²
