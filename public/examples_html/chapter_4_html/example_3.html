Example 4.3

In △BAC and △PRQ,

\(\frac{AB}{RQ} = \frac{3}{6} = \frac{1}{2}\)

\(\frac{BC}{QP} = \frac{6}{12} = \frac{1}{2}\)

\(\frac{CA}{PR} = \frac{3\sqrt{3}}{6\sqrt{3}} = \frac{1}{2}\)

Therefore, \(\frac{AB}{RQ} = \frac{BC}{QP} = \frac{CA}{PR}\)

By SSS similarity, we have △BAC ~ △QRP

\(\angle P = \angle C\) (since the corresponding parts of similar triangles are equal)

\(\angle P = \angle C = 180° - (\angle A + \angle B) = 180° - (90° + 60°)\)

\(\angle P = 180° - 150° = 30°\)
