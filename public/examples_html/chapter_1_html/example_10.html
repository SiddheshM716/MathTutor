Using vertical line test, determine which of the following curves (Fig.1.18(a), 1.18(b), 1.18(c), 1.18(d)) represent a function?

1

Understand the vertical line test. A curve represents a function if and only if no vertical line intersects the curve more than once.

2

Apply the vertical line test to each figure.

3

For Fig.1.18(a) and Fig.1.18(c), there exist vertical lines that intersect the curves at two points (P and Q). This violates the vertical line test.

For Fig.1.18(b) and Fig.1.18(d), every vertical line intersects the curve at most once, satisfying the vertical line test.

The curves in Fig.1.18(b) and Fig.1.18(d) represent functions, while Fig.1.18(a) and Fig.1.18(c) do not.
