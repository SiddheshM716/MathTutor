Example 1.5

The arrow diagram shows a relationship between the sets P and Q. Write the relation in (i) Set builder form (ii) Roster form (iii) What is the domain and range of R.

Solution

(i) Set builder form of R = {(x,y)|y = x-2, x ∈ P, y ∈ Q}

(ii) Roster form R = {(5,3),(6,4),(7,5)}

(iii) Domain of R = {5,6,7} and range of R = {3,4,5}

Let us consider the following example. Suppose A = {-3,-2,-1} and B = {1,2,3,4}. A relation from A to B is defined as a-b = 8 i.e., there is no pair(a,b) such that a-b=8. Thus R contain no element and so R=∅.

A relation which contains no element is called a "Null relation".
