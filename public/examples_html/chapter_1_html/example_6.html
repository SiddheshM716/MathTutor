Let \( X = \{1,2,3,4\} \) and \( Y = \{2,4,6,8,10\} \) and \( R = \{(1,2),(2,4),(3,6),(4,8)\} \). Show that \( R \) is a function and find its domain, co-domain and range.

1

To verify if \( R \) is a function, we check whether every element in \( X \) is mapped to exactly one element in \( Y \).

2

From the given relation \( R = \{(1,2),(2,4),(3,6),(4,8)\} \), we observe that each \( x \in X \) has a unique \( y \in Y \).

3

The domain of \( R \) is the set of all first elements in the ordered pairs, which is \( X = \{1,2,3,4\} \).

The co-domain is the set \( Y = \{2,4,6,8,10\} \).

The range is the set of all second elements in the ordered pairs, which is \( \{2,4,6,8\} \).

\( R \) is a function with domain \( \{1,2,3,4\} \), co-domain \( \{2,4,6,8,10\} \), and range \( \{2,4,6,8\} \).
