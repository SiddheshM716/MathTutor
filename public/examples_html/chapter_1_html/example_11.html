Let \( A = \{1, 2, 3, 4\} \) and \( B = \{2, 5, 8, 11, 14\} \) be two sets. Let \( f: A \rightarrow B \) be a function given by \( f(x) = 3x - 1 \). Represent this function:

(i) by arrow diagram

(ii) in a table form

(iii) as a set of ordered pairs

(iv) in a graphical form

1

Calculate \( f(x) \) for each \( x \in A \) using the formula \( f(x) = 3x - 1 \).

\[ f(1) = 3(1) - 1 = 2 \]

\[ f(2) = 3(2) - 1 = 5 \]

\[ f(3) = 3(3) - 1 = 8 \]

\[ f(4) = 3(4) - 1 = 11 \]

2

Represent the function as an arrow diagram.
