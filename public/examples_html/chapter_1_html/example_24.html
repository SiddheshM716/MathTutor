Example 1.24

Find x if g ∘ f ∘ f(x) = f ∘ g ∘ g(x), given f(x) = 3x + 1 and g(x) = x + 3.

Solution:

g ∘ f ∘ f(x) = g ∘ f(f(x))

= g ∘ f(3x + 1) = g(3(3x + 1) + 1) = g(9x + 4)

g(9x + 4) = (9x + 4) + 3 = 9x + 7

f ∘ g ∘ g(x) = f ∘ g(g(x))

= f ∘ g(x + 3) = f((x + 3) + 3) = f(x + 6)

f(x + 6) = 3(x + 6) + 1 = 3x + 19

These two quantities being equal, we get 9x + 7 = 3x + 19. Solving this equation we obtain x = 2.
