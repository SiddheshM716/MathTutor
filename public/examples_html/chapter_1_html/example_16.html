Verify the function \( h \) is one-one or not.

Find the height of a person if the length of his thigh bone is 50 cm.

Find the length of the thigh bone if the height of a person is 147.96 cm.

1

To check if \( h \) is one-one, assume \( h(b_1) = h(b_2) \).

Given \( h(b) = 2.47b + 54.10 \), we have:

\( 2.47b_1 + 54.10 = 2.47b_2 + 54.10 \).

2

Simplify the equation:

Subtract 54.10 from both sides:

\( 2.47b_1 = 2.47b_2 \).

3

Divide both sides by 2.47:

\( b_1 = b_2 \).

Thus, \( h \) is one-one.

The function \( h \) is one-one.

1

Given the length of the thigh bone \( b = 50 \) cm, use the function \( h(b) = 2.47b + 54.10 \).

2

Substitute \( b = 50 \) into the formula:

\( h(50) = 2.47 \times 50 + 54.10 \).

3

Calculate the height:

\( h(50) = 123.5 + 54.10 = 177.6 \) cm.

The height of the person is 177.6 cm.

1

Given the height \( h(b) = 147.96 \) cm, use the function \( h(b) = 2.47b + 54.10 \).

2

Set up the equation:

\( 2.47b + 54.10 = 147.96 \).

3

Solve for \( b \):

Subtract 54.10 from both sides:

\( 2.47b = 147.96 - 54.10 = 93.86 \).

Divide both sides by 2.47:

\( b = \frac{93.86}{2.47} = 38 \).

The length of the thigh bone is 38 cm.
