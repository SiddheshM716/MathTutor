Example 1.16

  1. Verify the function h is one-one or not.
  2. Find the height of a person if the length of his thigh bone is 50 cm.
  3. Find the length of the thigh bone if the height of a person is 147.96 cm.

Solution:

  1. To check if h is one-one, we assume that h(b₁) = h(b₂). Then we get, 2.47b₁ + 54.10 = 2.47b₂ + 54.10 ⇒ 2.47b₁ = 2.47b₂ ⇒ b₁ = b₂. Thus, h(b₁) = h(b₂) ⇒ b₁ = b₂. So, the function h is one-one.
  2. If the length of the thigh bone b = 50, then the height is h(50) = (2.47 × 50) + 54.10 = 177.6 cm.
  3. If the height of a person is 147.96 cm, then h(b) = 147.96 and so the length of the thigh bone is given by 2.47b + 54.10 = 147.96 ⇒ 2.47b = 147.96 − 54.10 = 93.86 ⇒ b = 93.86 / 2.47 = 38. Therefore, the length of the thigh bone is 38 cm.
