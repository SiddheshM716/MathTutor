If \( A = \{-2, -1, 0, 1, 2\} \) and \( f: A \rightarrow B \) is an onto function defined by \( f(x) = x^2 + x + 1 \), then find \( B \).

1

Evaluate \( f(x) = x^2 + x + 1 \) for each element in set \( A \).

2

Calculate \( f(-2) \):

\[ f(-2) = (-2)^2 + (-2) + 1 = 4 - 2 + 1 = 3 \]

3

Calculate \( f(-1) \):

\[ f(-1) = (-1)^2 + (-1) + 1 = 1 - 1 + 1 = 1 \]

4

Calculate \( f(0) \):

\[ f(0) = 0^2 + 0 + 1 = 0 + 0 + 1 = 1 \]

5

Calculate \( f(1) \):

\[ f(1) = 1^2 + 1 + 1 = 1 + 1 + 1 = 3 \]

6

Calculate \( f(2) \):

\[ f(2) = 2^2 + 2 + 1 = 4 + 2 + 1 = 7 \]

The range \( B \) is the set of all distinct outputs, so \( B = \{1, 3, 7\} \).
