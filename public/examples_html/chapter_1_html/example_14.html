Example 1.14

If \( A = \{-2, -1, 0, 1, 2\} \) and \( f: A \rightarrow B \) is an onto function defined by \( f(x) = x^2 + x + 1 \), then find \( B \).

Given \( A = \{-2, -1, 0, 1, 2\} \) and \( f(x) = x^2 + x + 1 \).

\( f(-2) = (-2)^2 + (-2) + 1 = 3 \);

\( f(-1) = (-1)^2 + (-1) + 1 = 1 \);

\( f(0) = 0^2 + 0 + 1 = 1 \);

\( f(1) = 1^2 + 1 + 1 = 3 \);

\( f(2) = 2^2 + 2 + 1 = 7 \).

Therefore, \( B = \{1, 3, 7\} \).
