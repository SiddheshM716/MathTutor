Using horizontal line test, determine which of the following functions are one-to-one.

The curves in Fig.1.35 (a) and Fig.1.35 (c) represent a one-one function as the horizontal lines meet the curves in only one point P.

The curve in Fig. 1.35 (b) does not represent a one-one function, since, the horizontal line intersects the curve at two points P and Q.

1

Understand the horizontal line test. A function is one-to-one if and only if no horizontal line intersects its graph more than once.

2

Analyze Fig.1.35 (a) and Fig.1.35 (c). In both cases, any horizontal line intersects the curve at only one point P.

3

Analyze Fig.1.35 (b). Here, at least one horizontal line intersects the curve at two points P and Q.

The functions represented by Fig.1.35 (a) and Fig.1.35 (c) are one-to-one, while the function represented by Fig.1.35 (b) is not one-to-one.
