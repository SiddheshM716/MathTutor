Prove that \(\frac{1 + \cot^2 \theta}{\csc \theta + 1} = \csc \theta\)

1

Recall the Pythagorean identity \(1 + \cot^2 \theta = \csc^2 \theta\). Substitute this into the numerator.

\[

\frac{1 + \cot^2 \theta}{\csc \theta + 1} = \frac{\csc^2 \theta}{\csc \theta + 1}

\]

2

Factor the numerator using the difference of squares formula \(a^2 - b^2 = (a + b)(a - b)\).

\[

\frac{\csc^2 \theta}{\csc \theta + 1} = \frac{(\csc \theta + 1)(\csc \theta - 1)}{\csc \theta + 1}

\]

3

Cancel out the common factor \((\csc \theta + 1)\) in the numerator and denominator.

\[

\frac{(\csc \theta + 1)(\csc \theta - 1)}{\csc \theta + 1} = \csc \theta - 1 + 1 = \csc \theta

\]

\(\csc \theta\)
