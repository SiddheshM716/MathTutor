Example 6.3

Prove that \(\frac{1 + \cot^2 \theta}{\csc \theta + 1} = \csc \theta\)

Solution:

\[ \frac{1 + \cot^2 \theta}{\csc \theta + 1} = \frac{\csc^2 \theta - 1 + 1}{\csc \theta + 1} = \frac{\csc^2 \theta}{\csc \theta + 1} \]

\[ = \frac{(\csc \theta + 1)(\csc \theta - 1)}{\csc \theta + 1} = \csc \theta - 1 + 1 = \csc \theta \]
