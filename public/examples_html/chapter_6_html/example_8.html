If \(\cos\theta + \sin\theta = \sqrt{2} \cos\phi\), then prove that \(\cos\theta - \sin\theta = \sqrt{2} \sin\phi\).

1

Start with the given equation \(\cos\theta + \sin\theta = \sqrt{2} \cos\phi\).

2

Square both sides to eliminate the square root and expand using the identity \((a + b)^2 = a^2 + 2ab + b^2\).

\((\cos\theta + \sin\theta)^2 = (\sqrt{2} \cos\phi)^2\)

3

Simplify both sides using trigonometric identities \(\cos^2\theta + \sin^2\theta = 1\) and \(2\sin\theta\cos\theta = \sin(2\theta)\).

\(\cos^2\theta + \sin^2\theta + 2\sin\theta\cos\theta = 2\cos^2\phi\)

\(1 + \sin(2\theta) = 2\cos^2\phi\)

4

Rearrange the equation to isolate \(\sin(2\theta)\).

\(\sin(2\theta) = 2\cos^2\phi - 1\)

5

Use the double-angle identity \(\cos(2\phi) = 2\cos^2\phi - 1\).

\(\sin(2\theta) = \cos(2\phi)\)

6

Express \(\cos(2\phi)\) in terms of sine using the co-function identity \(\cos(2\phi) = \sin\left(\frac{\pi}{2} - 2\phi\right)\).

\(\sin(2\theta) = \sin\left(\frac{\pi}{2} - 2\phi\right)\)

7

Solve for \(\theta\) and \(\phi\) by equating the arguments of the sine functions.

\(2\theta = \frac{\pi}{2} - 2\phi + 2\pi n\) (for any integer \(n\))

\(\theta + \phi = \frac{\pi}{4} + \pi n\)

8

Substitute \(\phi = \frac{\pi}{4} + \pi n - \theta\) back into the original equation to find \(\cos\theta - \sin\theta\).

\(\cos\theta - \sin\theta = \sqrt{2} \sin\left(\frac{\pi}{4} + \pi n - \theta\right)\)

\(\cos\theta - \sin\theta = \sqrt{2} \sin\phi\) is proven.
