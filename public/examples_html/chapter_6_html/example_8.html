Example 6.8

If \(\cos\theta + \sin\theta = \sqrt{2} \cos\phi\), then prove that \(\cos\theta - \sin\theta = \sqrt{2} \sin\phi\).

Solution:

Given, \(\cos\theta + \sin\theta = \sqrt{2} \cos\phi\).

Squaring both sides,

\((\cos\theta + \sin\theta)^2 = (\sqrt{2} \cos\phi)^2\)

\(\cos^2\theta + \sin^2\theta + 2\sin\theta\cos\theta = 2\cos^2\phi\)

\(2\cos^2\phi - \cos^2\theta - \sin^2\theta = 2\sin\theta\cos\theta\)

\(\cos^2\theta - \sin^2\theta = 2\sin\theta\cos\theta\)

\((\cos\theta + \sin\theta)(\cos\theta - \sin\theta) = 2\sin\theta\cos\theta\)

\(\cos\theta - \sin\theta = \frac{2\sin\theta\cos\theta}{\cos\theta + \sin\theta}\)

\(\cos\theta - \sin\theta = \sqrt{2} \sin\phi\)

Therefore, \(\cos\theta - \sin\theta = \sqrt{2} \sin\phi\).
