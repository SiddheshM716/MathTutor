Example 6.5

Prove that \(\frac{1 + \cos \theta}{1 - \cos \theta} = \csc \theta + \cot \theta\)

Solution:

\(\frac{1 + \cos \theta}{1 - \cos \theta} = \left(\frac{1 + \cos \theta}{1 - \cos \theta}\right) \times \left(\frac{1 + \cos \theta}{1 + \cos \theta}\right)\) [multiply numerator and denominator by the conjugate of \(1 - \cos \theta\)]

\(= \frac{(1 + \cos \theta)^2}{1 - \cos^2 \theta} = \frac{1 + 2\cos \theta + \cos^2 \theta}{\sin^2 \theta}\) [since \(\sin^2 \theta + \cos^2 \theta = 1\)]

\(= \frac{1}{\sin^2 \theta} + \frac{2\cos \theta}{\sin^2 \theta} + \frac{\cos^2 \theta}{\sin^2 \theta} = \csc^2 \theta + 2\csc \theta \cot \theta + \cot^2 \theta\)

\(= (\csc \theta + \cot \theta)^2\)

Therefore, \(\frac{1 + \cos \theta}{1 - \cos \theta} = \csc \theta + \cot \theta\)
