Prove that $\sin^2A \cos^2B + \cos^2A \sin^2B + \cos^2A \cos^2B + \sin^2A \sin^2B = 1$.

1

Rearrange the terms for easier grouping.

$\sin^2A \cos^2B + \cos^2A \sin^2B + \cos^2A \cos^2B + \sin^2A \sin^2B$

$= \sin^2A \cos^2B + \sin^2A \sin^2B + \cos^2A \sin^2B + \cos^2A \cos^2B$

2

Factor out common terms.

$= \sin^2A (\cos^2B + \sin^2B) + \cos^2A (\sin^2B + \cos^2B)$

3

Apply the Pythagorean identity $\sin^2\theta + \cos^2\theta = 1$.

$= \sin^2A (1) + \cos^2A (1)$

$= \sin^2A + \cos^2A$

$= 1$
