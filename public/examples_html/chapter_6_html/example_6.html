Prove that \(\frac{1}{\sin \theta \cos \theta} = \cot \theta\)

1

Start with the left-hand side (LHS) of the equation \(\frac{1}{\sin \theta \cos \theta}\).

2

Recall the trigonometric identity \(\cot \theta = \frac{\cos \theta}{\sin \theta}\).

3

Rewrite \(\frac{1}{\sin \theta \cos \theta}\) as \(\frac{\cos \theta}{\sin \theta \cos^2 \theta}\).

4

Simplify the expression using \(\cos^2 \theta = 1 - \sin^2 \theta\):

\[

\frac{\cos \theta}{\sin \theta (1 - \sin^2 \theta)} = \frac{\cos \theta}{\sin \theta \cos^2 \theta} = \frac{1}{\sin \theta \cos \theta}

\]

The simplified form matches the given equation, proving \(\frac{1}{\sin \theta \cos \theta} = \cot \theta\).
