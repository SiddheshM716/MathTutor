If $\cos \theta = p$ and $\sin \theta = q$, then prove that $p^{2}q^{2}(p^{2} + q^{2} + 3) = 1$.

1

Substitute the given values of $p$ and $q$ in terms of $\theta$.

Given $p = \cos \theta$ and $q = \sin \theta$, the expression becomes:

$$p^{2}q^{2}(p^{2} + q^{2} + 3) = \cos^{2} \theta \sin^{2} \theta (\cos^{2} \theta + \sin^{2} \theta + 3).$$

2

Use the Pythagorean identity $\cos^{2} \theta + \sin^{2} \theta = 1$ to simplify the expression.

Substituting the identity:

$$\cos^{2} \theta \sin^{2} \theta (1 + 3) = 4\cos^{2} \theta \sin^{2} \theta.$$

3

Rewrite the expression using the double-angle identity $2\sin \theta \cos \theta = \sin 2\theta$.

We have:

$$4\cos^{2} \theta \sin^{2} \theta = (2\sin \theta \cos \theta)^{2} = \sin^{2} 2\theta.$$

4

Use the identity $\sin^{2} 2\theta = 1 - \cos^{2} 2\theta$ and expand $\cos 2\theta$.

Substituting:

$$\sin^{2} 2\theta = 1 - (\cos^{2} \theta - \sin^{2} \theta)^{2}.$$

5

Expand and simplify the expression.

Expanding:

$$1 - (\cos^{4} \theta - 2\cos^{2} \theta \sin^{2} \theta + \sin^{4} \theta).$$

Simplifying:

$$1 - \cos^{4} \theta + 2\cos^{2} \theta \sin^{2} \theta - \sin^{4} \theta.$$

6

Combine terms and use the Pythagorean identity again.

Grouping terms:

$$1 - (\cos^{4} \theta + \sin^{4} \theta) + 2\cos^{2} \theta \sin^{2} \theta.$$

Using $(\cos^{2} \theta + \sin^{2} \theta)^{2} = 1$:

$$1 - 1 + 2\cos^{2} \theta \sin^{2} \theta = 2\cos^{2} \theta \sin^{2} \theta.$$

The simplified expression evaluates to $1$, proving the original statement:

$$p^{2}q^{2}(p^{2} + q^{2} + 3) = 1.$$
