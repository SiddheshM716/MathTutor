Show that the straight lines \( x - 2y + 3 = 0 \) and \( 6x + 3y + 8 = 0 \) are perpendicular.

1

Find the slope of the first line \( x - 2y + 3 = 0 \).

The slope \( m_1 \) of a line in the form \( ax + by + c = 0 \) is given by \( m_1 = -\frac{a}{b} \).

For \( x - 2y + 3 = 0 \), \( a = 1 \) and \( b = -2 \).

Thus, \( m_1 = -\frac{1}{-2} = \frac{1}{2} \).

2

Find the slope of the second line \( 6x + 3y + 8 = 0 \).

Using the same formula, for \( 6x + 3y + 8 = 0 \), \( a = 6 \) and \( b = 3 \).

Thus, \( m_2 = -\frac{6}{3} = -2 \).

3

Check if the product of the slopes equals \(-1\), which is the condition for perpendicularity.

Calculate \( m_1 \times m_2 = \frac{1}{2} \times (-2) = -1 \).

Since \( m_1 \times m_2 = -1 \), the two straight lines are perpendicular.
