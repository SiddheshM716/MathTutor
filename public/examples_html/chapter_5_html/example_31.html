Example 5.31

Find the slope of the line which is

(i) parallel to 3x - 7y = 11

(ii) perpendicular to 2x - 3y + 8 = 0

Solution:

(i) Given straight line is 3x - 7y = 11

=> 3x - 7y - 11 = 0

Slope m = -3 / -7 = 3/7

Since parallel lines have same slopes, slope of any line parallel to 3x - 7y = 11 is 3/7.

(ii) Given straight line is 2x - 3y + 8 = 0

Slope m = -2 / -3 = 2/3

Since product of slopes is -1 for perpendicular lines, slope of any line perpendicular to 2x - 3y + 8 = 0 is -3/2.
