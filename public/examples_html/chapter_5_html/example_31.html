Find the slope of the line which is

(i) parallel to \(3x - 7y = 11\)

(ii) perpendicular to \(2x - 3y + 8 = 0\)

1

For part (i), the given line is \(3x - 7y = 11\). Rewrite it in standard form \(Ax + By + C = 0\).

\(3x - 7y - 11 = 0\)

2

The slope \(m\) of a line in the form \(Ax + By + C = 0\) is given by \(m = -\frac{A}{B}\).

For the line \(3x - 7y - 11 = 0\), \(A = 3\) and \(B = -7\).

3

Calculate the slope.

\(m = -\frac{3}{-7} = \frac{3}{7}\)

Since parallel lines have the same slope, the slope of any line parallel to \(3x - 7y = 11\) is \(\frac{3}{7}\).

Final Answer (i): \(\frac{3}{7}\)

4

For part (ii), the given line is \(2x - 3y + 8 = 0\). Rewrite it in standard form \(Ax + By + C = 0\).

\(2x - 3y + 8 = 0\)

5

The slope \(m\) of a line in the form \(Ax + By + C = 0\) is given by \(m = -\frac{A}{B}\).

For the line \(2x - 3y + 8 = 0\), \(A = 2\) and \(B = -3\).

6

Calculate the slope.

\(m = -\frac{2}{-3} = \frac{2}{3}\)

7

The slope of a line perpendicular to another is the negative reciprocal of the original slope.

Thus, the perpendicular slope is \(-\frac{3}{2}\).

Final Answer (ii): \(-\frac{3}{2}\)
