Show that the straight lines \(2x + 3y - 8 = 0\) and \(4x + 6y + 18 = 0\) are parallel.

1

Find the slope of the first line \(2x + 3y - 8 = 0\).

The slope \(m\) of a line in the form \(ax + by + c = 0\) is given by \(m = -\frac{a}{b}\).

For \(2x + 3y - 8 = 0\), the slope \(m_1\) is:

\[

m_1 = -\frac{2}{3}

\]

2

Find the slope of the second line \(4x + 6y + 18 = 0\).

Using the same formula, the slope \(m_2\) is:

\[

m_2 = -\frac{4}{6} = -\frac{2}{3}

\]

3

Compare the slopes \(m_1\) and \(m_2\).

Since \(m_1 = m_2 = -\frac{2}{3}\), the two lines have equal slopes.

The lines \(2x + 3y - 8 = 0\) and \(4x + 6y + 18 = 0\) are parallel because their slopes are equal.
