Prove that the equation \( x^2(p^2 + q^2) + 2x(pr + qs) + r^2 + s^2 = 0 \) has no real roots. If \( ps = qr \), then show that the roots are real and equal.

1

Identify the coefficients of the quadratic equation \( x^2(p^2 + q^2) + 2x(pr + qs) + r^2 + s^2 = 0 \).

Here, \( a = p^2 + q^2 \), \( b = 2(pr + qs) \), \( c = r^2 + s^2 \).

2

Compute the discriminant \( \Delta \) using the formula \( \Delta = b^2 - 4ac \).

\[ \Delta = [2(pr + qs)]^2 - 4(p^2 + q^2)(r^2 + s^2) \]

3

Simplify the discriminant.

\[ \Delta = 4[p^2r^2 + 2pqrs + q^2s^2 - p^2r^2 - p^2s^2 - q^2r^2 - q^2s^2] \]

\[ \Delta = 4[-p^2s^2 + 2pqrs - q^2r^2] = -4[(ps - qr)^2] \leq 0 \]

Since \( \Delta < 0 \), the equation has no real roots.

4

If \( ps = qr \), substitute into the discriminant.

\[ \Delta = -4[(qr - qr)^2] = 0 \]

Thus, \( \Delta = 0 \), and the roots are real and equal.

The equation has no real roots unless \( ps = qr \), in which case the roots are real and equal.
