Example 3.28

Solve \(x^4 - 13x^2 + 42 = 0\)

Solution: Let \(x^2 = a\). Then, \(x^4 - 13x^2 + 42 = a^2 - 13a + 42 = (a - 7)(a - 6)\)

Given, \((a - 7)(a - 6) = 0\) we get, \(a = 7\) or \(6\).

Since \(a = x^2\), \(x^2 = 7\) then, \(x = \pm \sqrt{7}\) or \(x^2 = 6\) we get, \(x = \pm \sqrt{6}\)

Therefore the roots are \(x = \pm \sqrt{7}\) or \(x = \pm \sqrt{6}\)
