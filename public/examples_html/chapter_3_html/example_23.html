Find the zeroes of the quadratic expression \(x^2 + 8x + 12\).

1

Factor the quadratic expression \(x^2 + 8x + 12\).

The factored form is \(p(x) = (x + 2)(x + 6)\).

2

To find the zeroes, set \(p(x) = 0\) and solve for \(x\).

The equation becomes \((x + 2)(x + 6) = 0\).

3

Apply the zero-product property, which states that if \(ab = 0\), then \(a = 0\) or \(b = 0\).

Thus, \(x + 2 = 0\) or \(x + 6 = 0\).

Solving these gives \(x = -2\) and \(x = -6\).

The zeroes of the quadratic expression are \(-2\) and \(-6\).
