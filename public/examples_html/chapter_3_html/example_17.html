Find:

\[

\frac{x^2 + 20x + 36}{x^2 - 3x - 28} - \frac{x^2 + 12x + 4}{x^2 - 3x - 28}

\]

1

Since both fractions have the same denominator, subtract the numerators directly.

\[

\frac{(x^2 + 20x + 36) - (x^2 + 12x + 4)}{x^2 - 3x - 28}

\]

2

Simplify the numerator by combining like terms.

\[

\frac{8x + 32}{x^2 - 3x - 28}

\]

3

Factor the numerator and the denominator.

Numerator: \(8x + 32 = 8(x + 4)\)

Denominator: \(x^2 - 3x - 28 = (x - 7)(x + 4)\)

4

Cancel the common factor \((x + 4)\) from the numerator and denominator.

\[

\frac{8}{x - 7}

\]
