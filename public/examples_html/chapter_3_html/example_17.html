Example 3.17

Find:

\(\frac{x^2 + 20x + 36}{x^2 - 3x - 28} - \frac{x^2 + 12x + 4}{x^2 - 3x - 28}\)

Solution:

\[\frac{x^2 + 20x + 36}{x^2 - 3x - 28} - \frac{x^2 + 12x + 4}{x^2 - 3x - 28} = \frac{(x^2 + 20x + 36) - (x^2 + 12x + 4)}{x^2 - 3x - 28}\]

\[= \frac{8x + 32}{x^2 - 3x - 28} = \frac{8(x + 4)}{(x - 7)(x + 4)} = \frac{8}{x - 7}\]
