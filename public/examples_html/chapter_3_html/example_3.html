Example 3.3

Solve the following system of linear equations in three variables:

3x - 2y + z = 2 ...(1)

2x + 3y - z = 5 ...(2)

x + y + z = 6 ...(3)

Adding (1) and (2), 3x - 2y + z = 2

2x + 3y - z = 5

5x + y = 7 ...(4)

Adding (2) and (3), 2x + 3y - z = 5

x + y + z = 6

3x + 4y = 11 ...(5)

4 × (4) - (5), 20x + 4y = 28

3x + 4y = 11

17x = 17 ⇒ x = 1

Substituting x = 1 in (4), 5 + y = 7 ⇒ y = 2

Substituting x = 1, y = 2 in (3), 1 + 2 + z = 6 ⇒ z = 3

Therefore, x = 1, y = 2, z = 3
