(i) Find the values of 'k' for which the quadratic equation \( kx^2 - (8k + 4)x + 81 = 0 \) has real and equal roots?

(ii) Find the values of 'k' such that quadratic equation \( (k + 9)x^2 + (k + 1)x + 1 = 0 \) has no real roots?

1

For part (i), the quadratic equation \( kx^2 - (8k + 4)x + 81 = 0 \) has real and equal roots when the discriminant \( \Delta = 0 \).

2

The discriminant formula is \( \Delta = b^2 - 4ac \). Here, \( a = k \), \( b = -(8k + 4) \), and \( c = 81 \).

3

Substituting the values into the discriminant formula:

\[

[-(8k + 4)]^2 - 4(k)(81) = 0

\]

\[

64k^2 + 64k + 16 - 324k = 0

\]

\[

64k^2 - 260k + 16 = 0

\]

Divide by 4:

\[

16k^2 - 65k + 4 = 0

\]

Factorize:

\[

(16k - 1)(k - 4) = 0

\]

Thus, \( k = \frac{1}{16} \) or \( k = 4 \).

For part (i), the values of \( k \) are \( \boxed{\dfrac{1}{16}} \) and \( \boxed{4} \).

Step 1: For part (ii), the quadratic equation \( (k + 9)x^2 + (k + 1)x + 1 = 0 \) has no real roots when the discriminant \( \Delta < 0 \).

Step 2: The discriminant formula is \( \Delta = b^2 - 4ac \). Here, \( a = k + 9 \), \( b = k + 1 \), and \( c = 1 \).

Step 3: Substituting the values into the discriminant formula:

\[

(k + 1)^2 - 4(k + 9)(1) < 0

\]

\[

k^2 + 2k + 1 - 4k - 36 < 0

\]

\[

k^2 - 2k - 35 < 0

\]

Factorize:

\[

(k + 5)(k - 7) < 0

\]

Thus, \( -5 < k < 7 \).

For part (ii), the values of \( k \) lie in the interval \( \boxed{(-5, 7)} \).
