Example 3.26

Solve \(2x^2 - 2\sqrt{6}x + 3 = 0\)

Solution:

\(2x^2 - 2\sqrt{6}x + 3 = 2x^2 - \sqrt{6}x - \sqrt{6}x + 3\) (by splitting the middle term)

Now, equating the factors to zero we get,

\((\sqrt{2}x - \sqrt{3})(\sqrt{2}x - \sqrt{3}) = 0\)

\(\sqrt{2}x - \sqrt{3} = 0\)

\(\sqrt{2}x = \sqrt{3}\)

Therefore, the solution is \(x = \frac{\sqrt{3}}{\sqrt{2}}\).
