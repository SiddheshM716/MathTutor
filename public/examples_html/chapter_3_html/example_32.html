Example 3.32

Solve x2 + 2x − 2 = 0 by formula method.

Compare x2 + 2x − 2 = 0 with the standard form ax2 + bx + c = 0.

a = 1, b = 2, c = −2

x = −b ± √(b2 − 4ac)⁄2a

Substituting the values of a, b and c in the formula we get,

x = −2 ± √(4 − 4(1)(−2))⁄2 = −2 ± √12⁄2 = −1 ± √3

Therefore, x = −1 + √3, −1 − √3
