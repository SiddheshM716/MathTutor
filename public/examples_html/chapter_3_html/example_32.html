Solve \( x^2 + 2x - 2 = 0 \) by formula method.

1

Compare \( x^2 + 2x - 2 = 0 \) with the standard quadratic form \( ax^2 + bx + c = 0 \).

Here, \( a = 1 \), \( b = 2 \), and \( c = -2 \).

2

The quadratic formula is:

\[ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \]

3

Substitute \( a = 1 \), \( b = 2 \), and \( c = -2 \) into the formula:

\[ x = \frac{-2 \pm \sqrt{(2)^2 - 4(1)(-2)}}{2(1)} \]

\[ x = \frac{-2 \pm \sqrt{4 + 8}}{2} \]

\[ x = \frac{-2 \pm \sqrt{12}}{2} \]

Simplify \( \sqrt{12} \) to \( 2\sqrt{3} \):

\[ x = \frac{-2 \pm 2\sqrt{3}}{2} \]

Divide numerator and denominator by 2:

\[ x = -1 \pm \sqrt{3} \]

\( x = -1 + \sqrt{3} \) or \( x = -1 - \sqrt{3} \)
