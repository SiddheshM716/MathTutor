Find the excluded values of the following expressions (if any).

(i) \(\frac{x + 10}{8x}\)

(ii) \(\frac{7p + 2}{8p^2 + 13p + 5}\)

(iii) \(\frac{1}{x^2 + 1}\)

1

For \(\frac{x + 10}{8x}\), the expression is undefined when the denominator equals zero.

2

Set the denominator \(8x = 0\).

3

Solve for \(x\): \(x = 0\).

The excluded value is \(0\).

---

Step 1: For \(\frac{7p + 2}{8p^2 + 13p + 5}\), the expression is undefined when the denominator equals zero.

Step 2: Factor the denominator: \(8p^2 + 13p + 5 = (8p + 5)(p + 1)\).

Step 3: Set each factor to zero: \(8p + 5 = 0\) gives \(p = -\frac{5}{8}\), and \(p + 1 = 0\) gives \(p = -1\).

The excluded values are \(-\frac{5}{8}\) and \(-1\).

---

Step 1: For \(\frac{1}{x^2 + 1}\), the expression is undefined when the denominator equals zero.

Step 2: Analyze \(x^2 + 1\). Since \(x^2 \geq 0\) for all \(x\), \(x^2 + 1 \geq 1\).

Step 3: The denominator \(x^2 + 1\) is never zero for any real \(x\).

There are no real excluded values.
