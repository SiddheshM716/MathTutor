Example 3.35

Solve:

\( pqx^2 - (p + q)^2 x + (p + q)^2 = 0 \)

Solution:

Compare the coefficients with the standard form \( ax^2 + bx + c = 0 \):

\[ a = pq, \quad b = -(p + q)^2, \quad c = (p + q)^2 \]

Using the quadratic formula:

\[ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \]

Substituting the values:

\[ x = \frac{(p + q)^2 \pm \sqrt{(p + q)^4 - 4(pq)(p + q)^2}}{2pq} \]

Simplify the expression under the square root:

\[ = \frac{(p+q)^2 \pm \sqrt{(p+q)^2 [(p+q)^2 - 4pq]}}{2pq} \]

\[ = \frac{(p+q)^2 \pm \sqrt{(p+q)^2 (p^2 + q^2 - 2pq)}}{2pq} \]

\[ = \frac{(p+q)^2 \pm \sqrt{(p+q)^2 (p-q)^2}}{2pq} \]

\[ = \frac{(p+q)^2 \pm (p+q)(p-q)}{2pq} \]

\[ = \frac{(p+q)\{(p+q) \pm (p-q)\}}{2pq} \]

Therefore, the solutions are:

\[ x = \frac{p+q}{q} \quad \text{and} \quad x = \frac{p+q}{p} \]
