Example 3.57

If a matrix has 16 elements, what are the possible orders it can have?

We know that a matrix of order m x n, has m x n elements. Thus to find all possible orders of a matrix with 16 elements, we will find all ordered pairs of natural numbers whose product is 16.

Such ordered pairs are (1,16), (16,1), (4,4), (8,2), (2,8). Hence possible orders are 1x16, 16x1, 4x4, 2x8, 8x2.
