Example 3.29

Solve \(\frac{1}{x-1} + \frac{1}{x} = \frac{5}{2}\)

Solution: Let \(y = \frac{1}{x-1}\), then \(\frac{1}{x} = \frac{y}{y-1}\).

Therefore, \(\frac{1}{x-1} + \frac{1}{x} = \frac{5}{2}\) becomes \(y + \frac{y}{y-1} = \frac{5}{2}\).

This simplifies to \(2y^2 - 5y + 2 = 0\), then \(y = \frac{1}{2}\) or \(y = 2\).

For \(y = \frac{1}{2}\), we get \(2x = 2 - 1\) implies \(x = -1\).

For \(y = 2\), we get \(x = 2x - 2\) implies \(x = 2\).

Therefore, the roots are \(x = -1, 2\).
