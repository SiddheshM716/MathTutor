Solve \(3p^2 + \sqrt{5}p - 5 = 0\) by formula method.

1

Compare \(3p^2 + \sqrt{5}p - 5 = 0\) with the standard form \(ap^2 + bp + c = 0\).

Here, \(a = 3\), \(b = \sqrt{5}\), \(c = -5\).

2

Use the quadratic formula:

\[

p = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}

\]

3

Substitute the values of \(a\), \(b\), and \(c\) into the formula:

\[

p = \frac{-\sqrt{5} \pm \sqrt{(\sqrt{5})^2 - 4(3)(-5)}}{2(3)}

\]

Simplify the expression under the square root:

\[

p = \frac{-\sqrt{5} \pm \sqrt{5 + 60}}{6}

\]

\(p = \frac{-\sqrt{5} \pm \sqrt{65}}{6}\)
