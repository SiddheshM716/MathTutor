Example 3.34

Solve \(3p^2 + \sqrt{5}p - 5 = 0\) by formula method.

Solution: Compare \(3p^2 + \sqrt{5}p - 5 = 0\) with the standard form \(ap^2 + bp + c = 0\)

\(a = 3\), \(b = \sqrt{5}\), \(c = -5\).

\[ p = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \]

Substituting the values of \(a\), \(b\) and \(c\) in the formula we get,

\[ p = \frac{-\sqrt{5} \pm \sqrt{(\sqrt{5})^2 - 4(3)(-5)}}{2(3)} \]

Therefore, \(p = \frac{-\sqrt{5} \pm \sqrt{5 + 60}}{6}\)
