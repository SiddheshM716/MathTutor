Example 3.8

Solve:

\(\frac{1}{2x} + \frac{1}{4y} + \frac{1}{3z} = \frac{1}{4}\)

\(\frac{1}{5x} - \frac{1}{8y} + \frac{1}{5z} = \frac{2}{15}\)

\(\frac{1}{x} + \frac{1}{y} = 1\)

Solution:

Let \(\frac{1}{x} = p\), \(\frac{1}{y} = q\), \(\frac{1}{z} = r\)

The given equations are written as:

\(\frac{1}{2}p + \frac{1}{4}q + \frac{1}{3}r = \frac{1}{4}\)

\(\frac{1}{5}p - \frac{1}{8}q + \frac{1}{5}r = \frac{2}{15}\)

\(p + q = 1\)

By simplifying we get:

\(6p + 3q - 4r = 3\) (1)

\(15p - 3q + 60r = 32\) (3)

Substituting (2) in (1) and (3) we get:

\(15p - 4r = 3\) (4)

\(6p + 60r = 32\) reduces to \(3p + 30r = 16\) (5)

Solving (4) and (5):

\(11p - 4r = 3\)

\(15p + 150r = 80\)

Subtracting, we get:

\(-154r = -77\)

Thus, \(r = \frac{1}{2}\)

Substituting \(r = \frac{1}{2}\) in (4) we get:

\(15p - 2 = 3\)

Thus, \(p = \frac{1}{3}\)

From (2), \(q = 3p\) we get \(q = 1\)

Therefore, \(x = \frac{1}{p} = 3\), \(y = \frac{1}{q} = 1\), \(z = \frac{1}{r} = 2\).
