Example 3.16

Find

(i) \(\frac{14x^7}{3y} \div \frac{7x}{16y}\) (ii) \(\frac{x^2 - 16}{x + 4} \div \frac{x - 4}{x + 4}\) (iii) \(\frac{16x^2 - 20x - 3}{3x^2 - 2x - 1} \div \frac{8x^2 + 11x + 3}{3x^2 - 11x - 4}\)

Solution:

(i) \(\frac{14x^7}{3y} \div \frac{7x}{16y} = \frac{14x^7}{3y} \times \frac{16y}{7x} = 32x^6\) (ii) \(\frac{x^2 - 16}{x + 4} \div \frac{x - 4}{x + 4} = \frac{(x + 4)(x - 4)}{x + 4} \times \frac{x + 4}{x - 4} = x + 4\) (iii) \(\frac{16x^2 - 20x - 3}{3x^2 - 2x - 1} \div \frac{8x^2 + 11x + 3}{3x^2 - 11x - 4} = \frac{(8x + 3)(2x - 1)}{(3x + 1)(x - 1)} \times \frac{(3x + 1)(x - 4)}{(8x + 3)(x + 1)} = \frac{(2x - 1)(x - 4)}{(x - 1)(x + 1)} = \frac{2x^2 - 9x + 4}{x^2 - 1}\)
