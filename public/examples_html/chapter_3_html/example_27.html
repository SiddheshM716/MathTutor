Example 3.27

Solve \(2m^2 + 19m + 30 = 0\)

Solution:

\(2m^2 + 19m + 30 = 2m^2 + 4m + 15m + 30 = 2m(m + 2) + 15(m + 2)\)

\(= (m + 2)(2m + 15)\)

Equating the factors to zero we get,

\((m + 2)(2m + 15) = 0\)

\(\Rightarrow m + 2 = 0 \Rightarrow m = -2\) or \(2m + 15 = 0 \Rightarrow m = -\frac{15}{2}\)

Therefore the roots are \(-2, -\frac{15}{2}\).
