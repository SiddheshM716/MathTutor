Solve \(2m^2 + 19m + 30 = 0\).

1

Factor the quadratic equation \(2m^2 + 19m + 30\) by splitting the middle term.

\(2m^2 + 19m + 30 = 2m^2 + 4m + 15m + 30\).

2

Group the terms and factor out common terms.

\(2m^2 + 4m + 15m + 30 = 2m(m + 2) + 15(m + 2)\).

3

Factor out the common binomial \((m + 2)\).

\((m + 2)(2m + 15) = 0\).

4

Set each factor equal to zero and solve for \(m\).

\(m + 2 = 0 \Rightarrow m = -2\)

\(2m + 15 = 0 \Rightarrow m = -\frac{15}{2}\).

The roots are \(m = -2\) and \(m = -\frac{15}{2}\).
