Example 3.59

Find the value of a, b, c, d from the equation

\(\begin{bmatrix} a - b & 3 \\ 2a + c & 2a - b \end{bmatrix} = \begin{bmatrix} 1 & 5 \\ 0 & 2 \end{bmatrix}\)

Solution:

The given matrices are equal. Thus all corresponding elements are equal.

Therefore,

\(a - b = 1 \quad (1)\)

\(2a + c = 5 \quad (2)\)

\(2a - b = 0 \quad (3)\)

\(3c + d = 2 \quad (4)\)

From (3), \(2a - b = 0 \implies 2a = b \quad (5)\)

Put \(2a = b\) in equation (1), \(a - 2a = 1 \implies a = -1\)

Put \(a = -1\) in equation (5), \(2(-1) = b \implies b = -2\)

Put \(a = -1\) in equation (2), \(2(-1) + c = 5 \implies c = 7\)

Put \(c = 7\) in equation (4), \(3(7) + d = 2 \implies d = -19\)

Therefore, \(a = -1, b = -2, c = 7, d = -19\)
