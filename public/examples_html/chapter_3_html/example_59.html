Find the value of \( a, b, c, d \) from the equation:

\[

\begin{bmatrix}

a - b & 3 \\

2a + c & 2a - b

\end{bmatrix}

=

\begin{bmatrix}

1 & 5 \\

0 & 2

\end{bmatrix}

\]

1

Since the given matrices are equal, their corresponding elements must be equal. This gives the following equations:

\[

a - b = 1 \quad (1)

\]

\[

2a + c = 5 \quad (2)

\]

\[

2a - b = 0 \quad (3)

\]

\[

3c + d = 2 \quad (4)

\]

2

Solve equation (3) for \( b \):

\[

2a - b = 0 \implies b = 2a \quad (5)

\]

3

Substitute \( b = 2a \) into equation (1):

\[

a - 2a = 1 \implies -a = 1 \implies a = -1

\]

4

Substitute \( a = -1 \) into equation (5) to find \( b \):

\[

b = 2(-1) \implies b = -2

\]

5

Substitute \( a = -1 \) into equation (2) to find \( c \):

\[

2(-1) + c = 5 \implies -2 + c = 5 \implies c = 7

\]

6

Substitute \( c = 7 \) into equation (4) to find \( d \):

\[

3(7) + d = 2 \implies 21 + d = 2 \implies d = -19

\]

The values are \( a = -1 \), \( b = -2 \), \( c = 7 \), and \( d = -19 \).
