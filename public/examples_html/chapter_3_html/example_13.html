Reduce the rational expressions to their lowest form:

(i) \(\frac{x^2 - 9}{x^2 - 16}\)

(ii) \(\frac{x^2 + 8x + 16}{x - 3}\)

1

Factorize the numerator and denominator for part (i).

The numerator \(x^2 - 9\) is a difference of squares, and the denominator \(x^2 - 16\) is also a difference of squares.

2

Apply the difference of squares formula \(a^2 - b^2 = (a + b)(a - b)\).

For the numerator: \(x^2 - 9 = (x + 3)(x - 3)\).

For the denominator: \(x^2 - 16 = (x + 4)(x - 4)\).

3

Write the simplified form.

\(\frac{x^2 - 9}{x^2 - 16} = \frac{(x + 3)(x - 3)}{(x + 4)(x - 4)}\).

Final Answer (i): \(\frac{(x + 3)(x - 3)}{(x + 4)(x - 4)}\).

---

4

Factorize the numerator for part (ii).

The numerator \(x^2 + 8x + 16\) is a perfect square trinomial.

5

Apply the perfect square formula \(a^2 + 2ab + b^2 = (a + b)^2\).

For the numerator: \(x^2 + 8x + 16 = (x + 4)^2\).

6

Write the simplified form.

\(\frac{x^2 + 8x + 16}{x - 3} = \frac{(x + 4)^2}{x - 3}\).

Final Answer (ii): \(\frac{(x + 4)^2}{x - 3}\).
