Determine the nature of roots for the following quadratic equations:

(i) \(x^2 - x - 20 = 0\)

(ii) \(9x^2 - 24x + 16 = 0\)

(iii) \(2x^2 - 2x + 9 = 0\)

1

For equation (i) \(x^2 - x - 20 = 0\), identify coefficients \(a = 1\), \(b = -1\), \(c = -20\).

2

Use the discriminant formula \(\Delta = b^2 - 4ac\).

3

Calculate \(\Delta = (-1)^2 - 4(1)(-20) = 1 + 80 = 81\).

Since \(\Delta = 81 > 0\), the equation has real and unequal roots.

Step 1: For equation (ii) \(9x^2 - 24x + 16 = 0\), identify coefficients \(a = 9\), \(b = -24\), \(c = 16\).

Step 2: Use the discriminant formula \(\Delta = b^2 - 4ac\).

Step 3: Calculate \(\Delta = (-24)^2 - 4(9)(16) = 576 - 576 = 0\).

Since \(\Delta = 0\), the equation has real and equal roots.

Step 1: For equation (iii) \(2x^2 - 2x + 9 = 0\), identify coefficients \(a = 2\), \(b = -2\), \(c = 9\).

Step 2: Use the discriminant formula \(\Delta = b^2 - 4ac\).

Step 3: Calculate \(\Delta = (-2)^2 - 4(2)(9) = 4 - 72 = -68\).

Since \(\Delta = -68 < 0\), the equation has no real roots.
