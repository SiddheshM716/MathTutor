Find the square root of the following expressions:

(i) \( 16x^2 + 9y^2 - 24xy + 24x - 18y + 9 \)

(ii) \( (6x^2 + x - 1)(3x^2 + 2x - 1)(2x^2 + 3x + 1) \)

(iii)

\[ \sqrt{15x^2} + \left( \sqrt{3} + \sqrt{10} \right)x + \sqrt{2} \]

\[ \left[ \sqrt{5x^2} + \left( 2\sqrt{5} + 1 \right)x + 2 \right] \]

\[ \left[ \sqrt{3x^2} + \left( \sqrt{2} + 2\sqrt{3} \right)x + 2\sqrt{2} \right] \]

1

For part (i), observe that the expression can be written as a perfect square.

\[ 16x^2 + 9y^2 - 24xy + 24x - 18y + 9 = (4x - 3y + 3)^2 \]

2

Take the square root of the perfect square.

\[ \sqrt{(4x - 3y + 3)^2} = |4x - 3y + 3| \]

\( |4x - 3y + 3| \)

---

Step 1: For part (ii), factor each quadratic expression inside the product.

\[ 6x^2 + x - 1 = (3x - 1)(2x + 1) \]

\[ 3x^2 + 2x - 1 = (3x - 1)(x + 1) \]

\[ 2x^2 + 3x + 1 = (2x + 1)(x + 1) \]

Step 2: Multiply the factored forms and simplify.

\[ (6x^2 + x - 1)(3x^2 + 2x - 1)(2x^2 + 3x + 1) = (3x - 1)^2 (2x + 1)^2 (x + 1)^2 \]

Step 3: Take the square root of the simplified product.

\[ \sqrt{(3x - 1)^2 (2x + 1)^2 (x + 1)^2} = |(3x - 1)(2x + 1)(x + 1)| \]

\( |(3x - 1)(2x + 1)(x + 1)| \)

---

Step 1: For part (iii), factor each quadratic expression under the square roots.

\[ \sqrt{15x^2 + (\sqrt{3} + \sqrt{10})x + \sqrt{2}} = (\sqrt{5}x + 1)(\sqrt{3}x + \sqrt{2}) \]

\[ \sqrt{5x^2 + (2\sqrt{5} + 1)x + 2} = (\sqrt{5}x + 1)(x + 2) \]

\[ \sqrt{3x^2 + (\sqrt{2} + 2\sqrt{3})x + 2\sqrt{2}} = (x + 2)(\sqrt{3}x + \sqrt{2}) \]

Step 2: Multiply the factored forms and simplify.

\[ (\sqrt{5}x + 1)(\sqrt{3}x + \sqrt{2})(x + 2) \]

\( (\sqrt{5}x + 1)(\sqrt{3}x + \sqrt{2})(x + 2) \)
