Example 3.45

If \( \alpha \), \( \beta \) are the roots of the equation \( 3x^2 + 7x - 2 = 0 \), find the values of

(i) \( \frac{\beta}{\alpha} + \frac{\alpha}{\beta} \)

(ii) \( \left( \frac{\alpha}{\beta} + 1 \right) \left( \frac{\beta}{\alpha} + 1 \right) \)

Solution:

Given equation: \( 3x^2 + 7x - 2 = 0 \), where \( a = 3 \), \( b = 7 \), \( c = -2 \).

Since \( \alpha \), \( \beta \) are the roots of the equation:

\( \alpha + \beta = -\frac{b}{a} = -\frac{7}{3} \)

\( \alpha \beta = \frac{c}{a} = -\frac{2}{3} \)

(i) \( \frac{\beta}{\alpha} + \frac{\alpha}{\beta} = \frac{\alpha^2 + \beta^2}{\alpha \beta} = \frac{(\alpha + \beta)^2 - 2\alpha \beta}{\alpha \beta} = \frac{\left(-\frac{7}{3}\right)^2 - 2\left(-\frac{2}{3}\right)}{-\frac{2}{3}} = \frac{\frac{49}{9} + \frac{4}{3}}{-\frac{2}{3}} = \frac{\frac{49}{9} + \frac{12}{9}}{-\frac{2}{3}} = \frac{\frac{61}{9}}{-\frac{2}{3}} = -\frac{61}{6} \)

(ii) \( \left( \frac{\alpha}{\beta} + 1 \right) \left( \frac{\beta}{\alpha} + 1 \right) = \frac{\alpha}{\beta} \cdot \frac{\beta}{\alpha} + \frac{\alpha}{\beta} + \frac{\beta}{\alpha} + 1 = 1 + \frac{\alpha}{\beta} + \frac{\beta}{\alpha} + 1 = 2 + \left( \frac{\alpha}{\beta} + \frac{\beta}{\alpha} \right) = 2 + \left( -\frac{61}{6} \right) = 2 - \frac{61}{6} = \frac{12}{6} - \frac{61}{6} = -\frac{49}{6} \)
