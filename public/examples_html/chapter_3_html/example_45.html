If \( \alpha \), \( \beta \) are the roots of the equation \( 3x^2 + 7x - 2 = 0 \), find the values of

(i) \( \frac{\beta}{\alpha} + \frac{\alpha}{\beta} \)

(ii) \( \left( \frac{\alpha}{\beta} + 1 \right) \left( \frac{\beta}{\alpha} + 1 \right) \)

1

Identify the coefficients and root properties of the quadratic equation \( 3x^2 + 7x - 2 = 0 \). Here, \( a = 3 \), \( b = 7 \), \( c = -2 \).

2

Use the sum and product of roots formulas:

\[ \alpha + \beta = -\frac{b}{a} = -\frac{7}{3} \]

\[ \alpha \beta = \frac{c}{a} = -\frac{2}{3} \]

3

For part (i), rewrite the expression and simplify:

\[ \frac{\beta}{\alpha} + \frac{\alpha}{\beta} = \frac{\alpha^2 + \beta^2}{\alpha \beta} = \frac{(\alpha + \beta)^2 - 2\alpha \beta}{\alpha \beta} \]

Substitute the known values:

\[ \frac{\left(-\frac{7}{3}\right)^2 - 2\left(-\frac{2}{3}\right)}{-\frac{2}{3}} = \frac{\frac{49}{9} + \frac{4}{3}}{-\frac{2}{3}} \]

\[ = \frac{\frac{49}{9} + \frac{12}{9}}{-\frac{2}{3}} = \frac{\frac{61}{9}}{-\frac{2}{3}} = -\frac{61}{6} \]

Final Answer (i): \( -\frac{61}{6} \)

4

For part (ii), expand the expression and simplify:

\[ \left( \frac{\alpha}{\beta} + 1 \right) \left( \frac{\beta}{\alpha} + 1 \right) = 1 + \frac{\alpha}{\beta} + \frac{\beta}{\alpha} + 1 = 2 + \left( \frac{\alpha}{\beta} + \frac{\beta}{\alpha} \right) \]

Substitute the result from part (i):

\[ 2 + \left( -\frac{61}{6} \right) = 2 - \frac{61}{6} = \frac{12}{6} - \frac{61}{6} = -\frac{49}{6} \]

Final Answer (ii): \( -\frac{49}{6} \)
