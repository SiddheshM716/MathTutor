Example 3.53

Draw the graph of y = x2 + 4x + 3 and hence find the roots of

Scale: x-axis 1 cm = 1 unit, y-axis 1 cm = 2 units

Solution:

Step 1: Draw the graph of y = x2 + 4x + 3 by preparing the table of values as below:

x -2 -1 0 1 2
y -1 0 3 8 15

Step 2: To solve x2 + 2x + 1 = 0, subtract x2 + 2x + 1 = 0 from y = x2 + 4x + 3:

y = x2 + 4x + 3

0 = x2 + 2x + 1

The equation represents a straight line. Draw the graph of y = 2x + 2 by forming the table of values as below:

x -2 -1 0 1 2
y -2 0 2 4 6

Step 3: Observe that the graph of y = 2x + 2 does not intersect or touch the graph of the parabola y = x2 + 4x + 3.

Thus x2 + 2x + 1 = 0 has no real roots.
