Example 3.22

If \(9x^4 + 12x^3 + 28x^2 + ax + b\) is a perfect square, find the values of \(a\) and \(b\).

Solution:

\(3x^2 + 2x + 4\)
\(9x^4 + 12x^3 + 28x^2 + ax + b\) \((-)\)
\(9x^4\)
\(6x^2 + 2x\) \(\frac{12x^3 + 28x^2}{12x^3 + 4x^2}\) \((-)\)
\(6x^2 + 4x + 4\) \(\frac{24x^2 + ax + b}{24x^2 + 16x + 16}\) \((-)\)
0

Because the given polynomial is a perfect square:

\(a - 16 = 0\) and \(b - 16 = 0\)

Therefore \(a = 16\), \(b = 16\).
