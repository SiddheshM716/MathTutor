If \(9x^4 + 12x^3 + 28x^2 + ax + b\) is a perfect square, find the values of \(a\) and \(b\).

1

Assume the polynomial is a perfect square of the form \((3x^2 + px + q)^2\).

2

Expand \((3x^2 + px + q)^2\) to get:

\[

9x^4 + 6px^3 + (p^2 + 6q)x^2 + 2pqx + q^2

\]

3

Compare coefficients with the given polynomial \(9x^4 + 12x^3 + 28x^2 + ax + b\):

\[

6p = 12 \quad \Rightarrow \quad p = 2

\]

\[

p^2 + 6q = 28 \quad \Rightarrow \quad 4 + 6q = 28 \quad \Rightarrow \quad q = 4

\]

\[

2pq = a \quad \Rightarrow \quad 2 \cdot 2 \cdot 4 = 16 = a

\]

\[

q^2 = b \quad \Rightarrow \quad 16 = b

\]

\(a = 16\), \(b = 16\).
