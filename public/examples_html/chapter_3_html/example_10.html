Find the GCD of the polynomials \(2x^3 + 2x^2 - x + 2\) and \(2x^3 - 5x^2 + 5x - 3\).

1

Let \(f(x) = 2x^3 - 5x^2 + 5x - 3\) and \(g(x) = 2x^3 + 2x^2 - x + 2\). Divide \(f(x)\) by \(g(x)\).

\[

\begin{array}{r}

x^2 - x + 1 \\

\hline

2x^3 + 2x^2 - x + 2 \big| 2x^3 - 5x^2 + 5x - 3 \\

\underline{2x^3 + 2x^2 - 2x + 4} \\

-7x^2 + 7x - 7 \\

\end{array}

\]

2

Now, divide \(g(x)\) by the remainder \(-7x^2 + 7x - 7\) (or equivalently \(x^2 - x + 1\) after factoring out \(-7\)).

\[

\begin{array}{r}

x + 2 \\

\hline

x^2 - x + 1 \big| 2x^3 + 2x^2 - x + 2 \\

\underline{2x^3 - 2x^2 + 2x} \\

4x^2 - 3x + 2 \\

\underline{4x^2 - 4x + 4} \\

x - 2 \\

\end{array}

\]

3

The process stops here as the remainder \(x - 2\) does not divide the previous divisor \(x^2 - x + 1\) completely. The last non-zero remainder is \(x^2 - x + 1\).

\(\text{GCD}(2x^3 - 5x^2 + 5x - 3, 2x^3 + 2x^2 - x + 2) = x^2 - x + 1\).
