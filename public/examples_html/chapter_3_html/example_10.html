Example 3.10

Find the GCD of the polynomials \(2x^3 + 2x^2 - x + 2\) and \(2x^3 - 5x^2 + 5x - 3\).

Let \(f(x) = 2x^3 - 5x^2 + 5x - 3\) and \(g(x) = 2x^3 + 2x^2 - x + 2\).

Divide \(f(x)\) by \(g(x)\):

\[ \begin{array}{r} x^2 - x + 1 \\ \hline 2x^3 + 2x^2 - x + 2 \big| 2x^3 - 5x^2 + 5x - 3 \\ \underline{2x^3 + 2x^2 - 2x + 4} \\ -7x^2 + 7x - 7 \\ \end{array} \]

Now, divide \(g(x)\) by the new remainder \(x^2 - x + 1\):

\[ \begin{array}{r} x + 2 \\ \hline x^2 - x + 1 \big| 2x^3 + 2x^2 - x + 2 \\ \underline{2x^3 - 2x^2 + 2x} \\ 4x^2 - 3x + 2 \\ \underline{4x^2 - 4x + 4} \\ x - 2 \\ \end{array} \]

Here, we get zero remainder.

Therefore, \(\text{GCD}(2x^3 - 5x^2 + 5x - 3, 2x^3 + 2x^2 - x + 2) = x^2 - x + 1\).
