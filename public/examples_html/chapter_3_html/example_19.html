Example 3.19

Find the square root of the following expressions:

(i) \( 256(x - a)^8 (x - b)^4 (x - c)^{16} (x - d)^{20} \)

(ii) \( \frac{144 \, a^8 \, b^{12} \, c^{16}}{81 \, f^{12} \, g^4 \, h^{14}} \)

Solution:

(i) \( \sqrt{256(x - a)^8 (x - b)^4 (x - c)^{16} (x - d)^{20}} = 16 |(x - a)^4 (x - b)^2 (x - c)^8 (x - d)^{10}| \)

(ii) \( \sqrt{\frac{144a^8b^{12}c^{16}}{81f^{12}g^4h^{14}}} = \frac{4}{3} \left| \frac{a^4b^6c^8}{f^6g^2h^7} \right| \)
