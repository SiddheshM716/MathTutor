A die is rolled and a coin is tossed simultaneously. Find the probability that the die shows an odd number and the coin shows a head.

1

Determine the sample space (S) for the experiment where a die is rolled and a coin is tossed.

S = {1H, 1T, 2H, 2T, 3H, 3T, 4H, 4T, 5H, 5T, 6H, 6T}

n(S) = 12

2

Define event A as the die showing an odd number and the coin showing a head.

A = {1H, 3H, 5H}

n(A) = 3

3

Calculate the probability using the formula P(A) = n(A)/n(S).

P(A) = 3/12 = 1/4

\(\frac{1}{4}\)
