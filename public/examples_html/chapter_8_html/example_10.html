Example 8.10

Find the mean and variance of the first n natural numbers.

Solution

1. Calculating the Mean

\[ \text{Mean } \bar{x} = \frac{\text{Sum of numbers}}{\text{Count}} = \frac{1 + 2 + 3 + \cdots + n}{n} = \frac{\frac{n(n+1)}{2}}{n} = \frac{n+1}{2} \]

2. Calculating the Variance

Using the variance formula:

\[ \sigma^2 = \frac{\sum x_i^2}{n} - \left(\frac{\sum x_i}{n}\right)^2 \]

First term:

\[ \frac{\sum x_i^2}{n} = \frac{1^2 + 2^2 + \cdots + n^2}{n} = \frac{\frac{n(n+1)(2n+1)}{6}}{n} = \frac{(n+1)(2n+1)}{6} \]

Second term:

\[ \left(\frac{\sum x_i}{n}\right)^2 = \left(\frac{n+1}{2}\right)^2 = \frac{(n+1)^2}{4} \]

Combine terms:

\[ \sigma^2 = \frac{2n^2 + 3n + 1}{6} - \frac{n^2 + 2n + 1}{4} = \frac{4n^2 + 6n + 2 - 3n^2 - 6n - 3}{12} = \frac{n^2 - 1}{12} \]

Final Results

\[ \text{Mean} = \frac{n+1}{2}, \quad \text{Variance} = \frac{n^2 - 1}{12} \]

Additional: Standard Deviation for Grouped Data

For frequency distribution:

\[ \sigma = \sqrt{\frac{\sum f_i(x_i - \bar{x})^2}{N}} \quad \text{where } N = \sum f_i \]

Using deviations \( d_i = x_i - \bar{x} \):

\[ \sigma = \sqrt{\frac{\sum f_i d_i^2}{N}} \]
