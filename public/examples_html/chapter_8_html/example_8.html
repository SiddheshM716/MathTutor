Find the standard deviation of the following data: 7, 4, 8, 10, 11. Add 3 to all the values then find the standard deviation for the new values.

1

Arrange the given data in ascending order: 4, 7, 8, 10, 11. The number of data points \( n = 5 \).

2

Calculate the sum of the data points \( \Sigma x \) and the sum of their squares \( \Sigma x^2 \).

3

Compute the standard deviation using the formula:

\[

\sigma = \sqrt{\frac{\Sigma x^2}{n} - \left(\frac{\Sigma x}{n}\right)^2}

\]

\[

\sigma = \sqrt{\frac{350}{5} - \left(\frac{40}{5}\right)^2} = \sqrt{70 - 64} = \sqrt{6} \approx 2.45

\]

The standard deviation of the original data is \( \sqrt{6} \approx 2.45 \).
