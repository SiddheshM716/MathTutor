Example 8.9

Find the standard deviation of the data: 2, 3, 5, 7, 8. Then multiply each data by 4 and find the new standard deviation.

Solution

Part 1: Original Data

xᵢ xᵢ²
24
39
525
749
864
Σxᵢ = 25 Σxᵢ² = 151

Standard deviation calculation:

\[ \sigma = \sqrt{\frac{\Sigma x_i^2}{n} - \left(\frac{\Sigma x_i}{n}\right)^2} = \sqrt{\frac{151}{5} - \left(\frac{25}{5}\right)^2} \]

\[ = \sqrt{30.2 - 25} = \sqrt{5.2} \approx 2.28 \]

Part 2: Data Multiplied by 4

New data values: 8, 12, 20, 28, 32

xᵢ xᵢ²
864
12144
20400
28784
321024
Σxᵢ = 100 Σxᵢ² = 2416

Standard deviation calculation:

\[ \sigma = \sqrt{\frac{2416}{5} - \left(\frac{100}{5}\right)^2} = \sqrt{483.2 - 400} = \sqrt{83.2} \]

\[ = \sqrt{16 \times 5.2} = 4\sqrt{5.2} \approx 9.12 \]

Conclusion

When each data point is multiplied by 4, the standard deviation is also multiplied by 4 (from ≈2.28 to ≈9.12).
