Find the standard deviation of the data: 2, 3, 5, 7, 8. Then multiply each data by 4 and find the new standard deviation.

1

Calculate the standard deviation for the original data (2, 3, 5, 7, 8).

First, compute the sum of the data values and their squares:

\[

\Sigma x_i = 2 + 3 + 5 + 7 + 8 = 25

\]

\[

\Sigma x_i^2 = 2^2 + 3^2 + 5^2 + 7^2 + 8^2 = 4 + 9 + 25 + 49 + 64 = 151

\]

2

Use the standard deviation formula:

\[

\sigma = \sqrt{\frac{\Sigma x_i^2}{n} - \left(\frac{\Sigma x_i}{n}\right)^2}

\]

3

Substitute the values and compute:

\[

\sigma = \sqrt{\frac{151}{5} - \left(\frac{25}{5}\right)^2} = \sqrt{30.2 - 25} = \sqrt{5.2} \approx 2.28

\]

The standard deviation of the original data is approximately \( 2.28 \).

Step 1: Multiply each data point by 4 to get the new data: 8, 12, 20, 28, 32.

Compute the sum of the new data values and their squares:

\[

\Sigma x_i = 8 + 12 + 20 + 28 + 32 = 100

\]

\[

\Sigma x_i^2 = 8^2 + 12^2 + 20^2 + 28^2 + 32^2 = 64 + 144 + 400 + 784 + 1024 = 2416

\]

Step 2: Use the standard deviation formula again:

\[

\sigma = \sqrt{\frac{\Sigma x_i^2}{n} - \left(\frac{\Sigma x_i}{n}\right)^2}

\]

Step 3: Substitute the new values and compute:

\[

\sigma = \sqrt{\frac{2416}{5} - \left(\frac{100}{5}\right)^2} = \sqrt{483.2 - 400} = \sqrt{83.2} = 4\sqrt{5.2} \approx 9.12

\]

The standard deviation of the scaled data is approximately \( 9.12 \).
