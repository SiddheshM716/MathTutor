Calculations:

Total students N = 29

\[ \sigma = \sqrt{\frac{\sum f_i d_i^2}{N} - \left(\frac{\sum f_i d_i}{N}\right)^2} = \sqrt{\frac{240}{29} - \left(\frac{4}{29}\right)^2} \]

\[ = \sqrt{\frac{240 \times 29 - 16}{29 \times 29}} = \sqrt{\frac{6944}{841}} \approx 2.87 \]

Alternative Methods

1. Mean Method (for continuous data)

\[ \sigma = \sqrt{\frac{\sum f_i (x_i - \bar{x})^2}{N}} \]

Where xᵢ = class midpoint, fᵢ = class frequency

2. Step Deviation Method

For class interval width c:

\[ \sigma = c \times \sqrt{\frac{\sum f_i d_i^2}{N} - \left(\frac{\sum f_i d_i}{N}\right)^2} \]

Where \( d_i = \frac{x_i - A}{c} \)
