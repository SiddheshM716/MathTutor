The present value of a machine is ₹40,000 and its value depreciates each year by 10%. Find the estimated value of the machine in the 6th year.

1

The initial value of the machine is ₹40,000, and it depreciates at 10% per year. This means each year the value becomes 90% of its previous value.

2

The depreciation follows a geometric progression (G.P.) with first term \( a = 40,000 \) and common ratio \( r = 0.90 \).

3

To find the value in the 6th year, we calculate the 6th term of the G.P. using the formula:

\[ t_n = ar^{n-1} \]

4

Substitute \( n = 6 \), \( a = 40,000 \), and \( r = 0.90 \):

\[ t_6 = 40,000 \times (0.90)^{5} \]

5

Compute the calculation:

\[ t_6 = 40,000 \times 0.59049 = 23,619.6 \]

The estimated value of the machine in the 6th year is ₹23,619.60.
