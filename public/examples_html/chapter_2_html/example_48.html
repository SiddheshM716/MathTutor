Example 2.48

How many terms of the series 1 + 4 + 16 + ... make the sum 1365?

Solution: Let n be the number of terms to be added to get the sum 1365.

a = 1, r = 4 > 1

S = 1365 ⇒ \(\frac{a(r^n - 1)}{r - 1} = 1365\)

\(\frac{1(4^n - 1)}{4 - 1} = 1365\) ⇒ \((4^n - 1) = 4095\)

\(4^n = 4096\) ⇒ \(4^n = 4^6\)

n = 6
