Example 2.25

Find the 15th, 24th and nth term (general term) of an A.P. given by 3, 15, 27, 39,...

Solution: We have, first term = a = 3 and common difference = d = 15 - 3 = 12.

We know that nth term (general term) of an A.P. with first term a and common difference d is given by tn = a + (n - 1)d

t15 = a + (15 - 1)d = a + 14d = 3 + 14(12) = 171

t24 = a + (24 - 1)d = a + 23d = 3 + 23(12) = 279

The nth (general term) term is given by tn = a + (n - 1)d

Thus, tn = 3 + (n - 1)12

tn = 12n - 9
