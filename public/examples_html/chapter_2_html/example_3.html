Example 2.3

Show that the square of an odd integer is of the form 4q+1, for some integer q.

Solution Let x be any odd integer. Since any odd integer is one more than an even integer, we have x = 2k + 1, for some integer k.

x² = (2k + 1)²

= 4k² + 4k + 1

= 4k(k + 1) + 1

= 4q + 1, where q = k(k + 1) is some integer.
