Find the sum of 0.40 + 0.43 + 0.46 + ... + 1.

1

Identify the given values.

Given, first term \( a = 0.40 \), last term \( l = 1 \), and common difference \( d = 0.43 - 0.40 = 0.03 \).

2

Calculate the number of terms \( n \) using the formula:

\[ n = \frac{l - a}{d} + 1 \]

Substitute the values:

\[ n = \frac{1 - 0.40}{0.03} + 1 = \frac{0.60}{0.03} + 1 = 20 + 1 = 21 \]

3

Compute the sum of the arithmetic progression using the formula:

\[ S_n = \frac{n}{2} (a + l) \]

Substitute \( n = 21 \), \( a = 0.40 \), and \( l = 1 \):

\[ S_{21} = \frac{21}{2} [0.40 + 1] = \frac{21}{2} \times 1.40 = 10.5 \times 1.40 = 14.7 \]

The sum of the series is \( \boxed{14.7} \).
