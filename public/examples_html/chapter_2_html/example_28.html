If \( l \), \( m \) and \( n \) terms of an A.P. are \( x \), \( y \), \( z \) respectively, then show that:

(i) \( x(m - n) + y(n - l) + z(l - m) = 0 \)

(ii) \( (x - y)n + (y - z)l + (z - x)m = 0 \)

1

Let \( a \) be the first term and \( d \) be the common difference of the A.P. Given the \( l \)-th, \( m \)-th, and \( n \)-th terms:

\( a + (l - 1)d = x \) (1)

\( a + (m - 1)d = y \) (2)

\( a + (n - 1)d = z \) (3)

2

For part (i), expand \( x(m - n) + y(n - l) + z(l - m) \) using equations (1), (2), and (3):

\( x(m - n) + y(n - l) + z(l - m) = a[(m - n) + (n - l) + (l - m)] + d[(m - n)(l - 1) + (n - l)(m - 1) + (l - m)(n - 1)] \)

3

Simplify the expression:

\( = a[0] + d[lm - ln - m + n + mn - lm - n + l + ln - mn - l + m] \)

\( = a(0) + d(0) = 0 \)

\( x(m - n) + y(n - l) + z(l - m) = 0 \)

Step 1: For part (ii), subtract equations (1)-(2), (2)-(3), and (3)-(1):

\( x - y = (l - m)d \)

\( y - z = (m - n)d \)

\( z - x = (n - l)d \)

Step 2: Substitute these into \( (x - y)n + (y - z)l + (z - x)m \):

\( (x - y)n + (y - z)l + (z - x)m = [(l - m)n + (m - n)l + (n - l)m]d \)

Step 3: Simplify the expression:

\( = [ln - mn + lm - nl + nm - lm]d = 0 \)

\( (x - y)n + (y - z)l + (z - x)m = 0 \)
