Example 2.57

  1. Find the sum of \(1^2 + 2^2 + 3^2 + \dots + 16^2\)

    2. Solution: \(1^2 + 2^2 + 3^2 + \dots + 16^2 = \frac{16 \times (16 + 1) \times (2 \times 16 + 1)}{6} = 1496\)

  2. Find the sum of \(9^2 + 10^2 + \dots + 21^2\)

    3. Solution: \(9^2 + 10^2 + \dots + 21^2 = \left(1^2 + 2^2 + \dots + 21^2\right) - \left(1^2 + 2^2 + \dots + 8^2\right)\)

    \(\left(\frac{21 \times (21 + 1) \times (2 \times 21 + 1)}{6}\right) - \left(\frac{8 \times (8 + 1) \times (2 \times 8 + 1)}{6}\right) = 3311 - 204 = 3107\)
