Example 2.54

Find the value of

(i) 1 + 2 + 3 + 4 + ... + 50

Using, 1 + 2 + 3 + ... + n = \(\frac{n(n + 1)}{2}\)

1 + 2 + 3 + ... + 50 = \(\frac{50 \times (50 + 1)}{2}\) = 1275

(ii) 16 + 17 + 18 + ... + 75

16 + 17 + 18 + ... + 75 = (1 + 2 + 3 + ... + 75) - (1 + 2 + 3 + ... + 15)

= \(\frac{75 \times (75 + 1)}{2} - \frac{15 \times (15 + 1)}{2}\)

= 2850 - 120 = 2730
