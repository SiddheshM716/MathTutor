Find the value of:

(i) \(1 + 2 + 3 + 4 + \dots + 50\)

(ii) \(16 + 17 + 18 + \dots + 75\)

1

For part (i), use the formula for the sum of the first \(n\) natural numbers:

\[

1 + 2 + 3 + \dots + n = \frac{n(n + 1)}{2}

\]

2

Substitute \(n = 50\) into the formula:

\[

1 + 2 + 3 + \dots + 50 = \frac{50 \times (50 + 1)}{2}

\]

3

Calculate the result:

\[

\frac{50 \times 51}{2} = \frac{2550}{2} = 1275

\]

\(1275\)

Step 1: For part (ii), express the sum as the difference of two sums:

\[

16 + 17 + 18 + \dots + 75 = (1 + 2 + 3 + \dots + 75) - (1 + 2 + 3 + \dots + 15)

\]

Step 2: Apply the sum formula to both terms:

\[

= \frac{75 \times (75 + 1)}{2} - \frac{15 \times (15 + 1)}{2}

\]

Step 3: Calculate each term and subtract:

\[

= \frac{75 \times 76}{2} - \frac{15 \times 16}{2} = 2850 - 120 = 2730

\]

\(2730\)
