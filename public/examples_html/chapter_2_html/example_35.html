Example 2.35

In an A.P. the sum of first n terms is \( \frac{5n^2 + 3n}{2} \). Find the 17th term.

The 17th term can be obtained by subtracting the sum of first 16 terms from the sum of first 17 terms.

\[ S_{17} = \frac{5 \times 17^2 + 3 \times 17}{2} = \frac{1445 + 51}{2} = 748 \]

\[ S_{16} = \frac{5 \times 16^2 + 3 \times 16}{2} = \frac{1280 + 48}{2} = 664 \]

Now, \( t_{17} = S_{17} - S_{16} = 748 - 664 = 84 \).
