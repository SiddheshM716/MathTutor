Find the number of terms in the A.P. 3, 6, 9, 12,..., 111.

1

Identify the given values from the arithmetic progression (A.P.).

First term \( a = 3 \), common difference \( d = 6 - 3 = 3 \), and last term \( l = 111 \).

2

Use the formula for the number of terms \( n \) in an A.P.:

\[ n = \frac{l - a}{d} + 1 \]

3

Substitute the known values into the formula and calculate:

\[ n = \frac{111 - 3}{3} + 1 = \frac{108}{3} + 1 = 36 + 1 = 37 \]

The A.P. contains \(\boxed{37}\) terms.
