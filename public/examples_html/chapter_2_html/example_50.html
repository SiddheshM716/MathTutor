Example 2.50

Find the rational form of the number 0.6666...

Solution: We can express the number 0.6666... as follows:

0.6666... = 0.6 + 0.06 + 0.006 + 0.0006 + ...

We now see that numbers 0.6, 0.06, 0.006... form a G.P. whose first term \( a = 0.6 \) and common ratio \( r = 0.1 \). Also \( -1 < r = 0.1 < 1 \).

Using the infinite G.P. formula, we have:

\[ 0.6666... = \frac{0.6}{1 - 0.1} = \frac{0.6}{0.9} = \frac{2}{3} \]

Thus the rational number equivalent of 0.6666... is \( \frac{2}{3} \).
