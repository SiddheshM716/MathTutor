Example 2.51

Find the sum to n terms of the series 5 + 55 + 555 + ---

Solution The series is neither Arithmetic nor Geometric series. So it can be split into two series and then find the sum.

5 + 55 + 555 + --- + n terms = 5[1 + 11 + 111 + --- + n terms]

= \(\frac{5}{9}\)[9 + 99 + 999 + --- + n terms]

= \(\frac{5}{9}\)[(10 - 1) + (100 - 1) + (1000 - 1) + --- + n terms]

= \(\frac{5}{9}\)[(10 + 100 + 1000 + --- + n terms) - n]

= \(\frac{5}{9}\left[\frac{10(10^n - 1)}{10 - 1} - n\right]\)
