Which of the following sequences form a Geometric Progression?

(i) 7, 14, 21, 28, ...

(ii) \(\frac{1}{2}\), 1, 2, 4, ...

(iii) 5, 25, 50, 75, ...

1

Check if the ratio between consecutive terms is constant for each sequence.

2

For sequence (i) 7, 14, 21, 28, ..., calculate the ratios:

\[ \frac{14}{7} = 2, \quad \frac{21}{14} = 1.5, \quad \frac{28}{21} \approx 1.333 \]

Since the ratios are not equal, this is not a Geometric Progression.

3

For sequence (ii) \(\frac{1}{2}\), 1, 2, 4, ..., calculate the ratios:

\[ \frac{1}{\frac{1}{2}} = 2, \quad \frac{2}{1} = 2, \quad \frac{4}{2} = 2 \]

Since the ratios are equal, this is a Geometric Progression with common ratio \( r = 2 \).

4

For sequence (iii) 5, 25, 50, 75, ..., calculate the ratios:

\[ \frac{25}{5} = 5, \quad \frac{50}{25} = 2, \quad \frac{75}{50} = 1.5 \]

Since the ratios are not equal, this is not a Geometric Progression.

Only sequence (ii) \(\frac{1}{2}\), 1, 2, 4, ... is a Geometric Progression.
