A mosaic is designed in the shape of an equilateral triangle, 12 ft on each side. Each tile in the mosaic is in the shape of an equilateral triangle of 12 inch side. The tiles are alternate in colour as shown in the figure. Find the number of tiles of each colour and total number of tiles in the mosaic.

1

The mosaic is an equilateral triangle with side length 12 ft, and each tile is an equilateral triangle with side length 12 inches (1 ft). Therefore, the mosaic will have 12 rows.

2

The number of white tiles in each row forms an Arithmetic Progression (AP): 1, 2, 3, ..., 12. The sum of an AP is given by the formula:

\[

S_n = \frac{n}{2} \times (a_1 + a_n)

\]

where \( n \) is the number of terms, \( a_1 \) is the first term, and \( a_n \) is the last term.

3

For white tiles:

\[

S_{12} = \frac{12}{2} \times (1 + 12) = 6 \times 13 = 78

\]

4

The number of blue tiles in each row forms another AP: 0, 1, 2, ..., 11. Using the same formula:

\[

S_{12} = \frac{12}{2} \times (0 + 11) = 6 \times 11 = 66

\]

5

The total number of tiles is the sum of white and blue tiles:

\[

78 + 66 = 144

\]

Number of white tiles = 78, Number of blue tiles = 66, Total number of tiles = 144.
